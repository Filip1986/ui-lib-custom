# Demo Theme Editor

This sidebar panel provides live, in-demo customization for variants, shape, density, mode, colors, typography, presets, and exports.

## Quick Use

- Click the floating gear button to open the editor.
- Use the Variant/Shape/Density/Mode pills to adjust global settings.
- Edit colors to apply updates instantly via CSS variables.
- Enter heading/body fonts and blur the input to apply.
- Save presets for reuse, apply them later, or delete.
- Export JSON or copy CSS for sharing.

## Implementation

- Component: `projects/demo/src/app/shared/theme-editor/theme-editor.component.ts`
- Services: `ThemeConfigService`, `ThemeEditorService`, `ThemePresetService`

No additional dependencies are required.
