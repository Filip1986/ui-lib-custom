Skeleton

Content placeholder displayed while real content is loading. Combine shapes to mock realistic page layouts.

Shapes

Use shape="rectangle" (default) or shape="circle".

Card Placeholder

Combine rectangle and circle skeletons to simulate a card layout.

List Placeholder

Repeat a row pattern to mock a list of items.

@for (item of [1, 2, 3, 4]; track item) {
}

Variants

Three design variants: material, bootstrap, minimal.

@for (variant of variants; track variant) {

{{ variant }}

}

Animation

Use animation="wave" (default) or animation="none" to disable the shimmer.

wave (default)

none

Custom Sizes

Width, height, and size accept any CSS dimension. The size shorthand sets both dimensions at once.

Custom Border Radius

Override the default radius with the borderRadius input.

API

Input Type Default Description
shape 'rectangle' | 'circle' 'rectangle' Shape of the placeholder.
width string '100%' CSS width value. Overridden by size.
height string '1rem' CSS height value. Overridden by size.
size string | null null Sets both width and height. Useful for circles.
borderRadius string | null null Custom border-radius, overrides shape default.
animation 'wave' | 'none' 'wave' Shimmer animation type.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService.
styleClass string | null null Additional CSS classes on the host element.
