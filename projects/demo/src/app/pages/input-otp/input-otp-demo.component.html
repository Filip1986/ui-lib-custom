InputOtp

Individual-cell OTP entry with paste support, keyboard navigation, mask mode, integer-only mode, three size tokens, and full Angular forms integration.

a. Basic

Four-cell OTP with ngModel binding and live value preview.

Value: {{ formatValue(basicValue) }}

b. Mask

Password-style dots for sensitive codes such as PINs.

c. Integer Only

Restricts each cell to digits 0–9, rejecting all other keystrokes.

Value: {{ formatValue(integerValue) }}

d. Sizes

Three size tokens — sm, md (default), lg.

e. Filled

Filled background variant for surfaces that need stronger contrast.

f. Disabled

All cells are non-interactive and rendered with reduced opacity.

g. Invalid

Error border applied to all cells when validation fails.

h. Read Only

Cells display a value but do not accept new input.

i. Reactive Forms

Six-digit numeric code with Validators.required and Validators.minLength(6). Invalid styling activates on touch.
