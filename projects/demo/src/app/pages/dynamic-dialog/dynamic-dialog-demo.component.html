DynamicDialog

Programmatically open any Angular component inside a configurable modal shell. No template tag required — just inject DialogService and call open().

Basic Usage

Call dialogService.open(Component, config) from any class that injects DialogService. 

const ref = this.dialogService.open(MyComponent, {
  header: 'Welcome',
  width: '32rem',
});

Passing Data to the Guest Component

Pass arbitrary data through config.data. The guest component reads it by injecting DYNAMIC_DIALOG_CONFIG.

// Caller
const ref = this.dialogService.open(DataDialogContentComponent, {
  header: 'User Details',
  data: { userId: 42, name: 'Filip Luca' },
});
ref.onClose.subscribe((result) => console.log(result));

// Guest component
config = inject(DYNAMIC_DIALOG_CONFIG);
ref    = inject(DynamicDialogRef);

this.userId = (this.config.data as any).userId;
this.ref.close({ confirmed: true });

Fixed Height & Scrollable Content

Set height on the config to give the dialog a fixed viewport height. The content area scrolls automatically. 

this.dialogService.open(LongContentComponent, {
  header: 'Scrollable',
  width: '38rem',
  height: '50vh',
});

Design Variants

Pass variant to override the inherited theme setting for a single dialog. 

this.dialogService.open(MyComponent, { variant: 'bootstrap' });

Viewport Positions

Use position to anchor the dialog to the center or any edge of the viewport. 

this.dialogService.open(MyComponent, { position: 'top' });
