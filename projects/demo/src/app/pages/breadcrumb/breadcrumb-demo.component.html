Breadcrumb

A breadcrumb displays a hierarchical navigation trail, helping users understand where they are in an application and navigate back to parent sections.

Basic

A simple breadcrumb with external URL links.

With Home

Use the home input to pin a home item at the start.

Use routerLink to navigate within the Angular application.

API

Input Type Default Description
items BreadcrumbItem[] [] The navigation items to display.
home BreadcrumbItem | null null Optional home item prepended before items.
variant 'material' | 'bootstrap' | 'minimal' | null null Design-system variant; falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Size token controlling font size and spacing.
styleClass string | null null Extra CSS class appended to host.
ariaLabel string 'Breadcrumb' Accessible label for the <nav> landmark.

Outputs

Output Type Description
itemClick BreadcrumbItemClickEvent Emitted when any breadcrumb item is clicked or activated via keyboard.

BreadcrumbItem interface

Property Type Description
label string? Display text for the item.
url string? External href URL.
routerLink string | string[] Angular RouterLink for in-app navigation.
icon string? Icon class name rendered before the label.
target string? Anchor target (e.g. _blank).
disabled boolean? When true, the item is non-interactive.
styleClass string? Extra CSS class on the item.
command (event: BreadcrumbItemClickEvent) => void Callback invoked when the item is activated.
