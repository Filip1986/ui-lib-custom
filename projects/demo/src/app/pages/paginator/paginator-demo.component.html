Paginator

Paginator provides page-based navigation for any list or data set. It supports three visual variants, three sizes, optional first/last buttons, a rows-per-page dropdown, a jump-to-page input, and content projection slots.

Variants

Three visual styles: material (default), bootstrap, and minimal.

Sizes

Control button dimensions with the size input: sm, md (default), lg.

Current Page Report

Enable showCurrentPageReport to display a summary. Placeholders: {currentPage}, {totalPages}, {first}, {last}, {rows}, {totalRecords}.

Rows Per Page

Pass an array of numbers to rowsPerPageOptions to show a dropdown.

Jump To Page

Enable showJumpToPageInput to show a numeric input. Type a page number and press Enter.

Without First / Last Icons

Page Links Off (arrows only)
