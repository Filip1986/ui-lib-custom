Severities

Four severity levels map to distinct colour palettes and icons.

Multiple Toasts

Multiple toasts can be displayed simultaneously and stack in the container.

Sticky & Non-Closable

Set sticky: true to prevent auto-dismissal. Set closable: false to hide the close button.

Auto-dismiss Duration

Control how long a toast stays visible with the life property (milliseconds). Per-message life overrides the container default.

Position & Variant

Choose a position and design variant, then trigger a toast to see it in action.

Position
@for (pos of positions; track pos) { {{ pos }} }
Variant
@for (v of variants; track v) { {{ v }} }

Keyed Containers

Use the key input on both the container and the message to route toasts to a specific container. Useful for displaying toasts in different screen regions simultaneously.
