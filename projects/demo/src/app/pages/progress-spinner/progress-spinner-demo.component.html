Drop in <ui-lib-progress-spinner /> with no inputs for a default medium-sized, theme-variant spinner.

The size input accepts 'sm', 'md', or 'lg'.

Override the global theme variant with the variant input. 'material' cycles through multiple colours, 'bootstrap' uses a solid primary blue, and 'minimal' uses a muted grey.
