ProgressSpinner

An animated circular spinner that signals an indeterminate loading state. Uses a CSS stroke-dashoffset animation on an SVG circle combined with a continuous rotation to produce the characteristic "arc chase" effect. Three design variants, three sizes, and full CSS-variable theming.

Component Misc A11y

Basic

Drop in <ui-lib-progress-spinner /> with no inputs for a default medium-sized, theme-variant spinner. 

<ui-lib-progress-spinner />

Sizes

The size input accepts 'sm', 'md', or 'lg'. 

<ui-lib-progress-spinner size="sm" />
<ui-lib-progress-spinner size="md" />
<ui-lib-progress-spinner size="lg" />

Variants

Override the global theme variant with the variant input. 'material' cycles through multiple colours, 'bootstrap' uses a solid primary blue, and 'minimal' uses a muted grey.

material
bootstrap
minimal 
<ui-lib-progress-spinner variant="material" />
<ui-lib-progress-spinner variant="bootstrap" />
<ui-lib-progress-spinner variant="minimal" />

Animation Duration

Control the rotation speed with animationDuration. Accepts any CSS <time> value.

Slow (4s)
Normal (2s)
Fast (750ms) 
<ui-lib-progress-spinner animationDuration="4s" />
<ui-lib-progress-spinner animationDuration="2s" />
<ui-lib-progress-spinner animationDuration="750ms" />

Stroke Width & Fill

strokeWidth controls the arc thickness (SVG units). fill sets the interior fill colour of the circle. 

<ui-lib-progress-spinner strokeWidth="1" />
<ui-lib-progress-spinner strokeWidth="4" />
<ui-lib-progress-spinner strokeWidth="6" fill="#e0e7ff" />

Loading Overlay Pattern

Combine ProgressSpinner with a semi-transparent overlay to block content during async operations.

Content that is loading…

More content here.

<div class="loading-wrapper">
  <div class="content">…</div>
  <div class="overlay">
    <ui-lib-progress-spinner size="lg" />
  </div>
</div>

API

Inputs

Input Type Default Description
strokeWidth string '2' SVG stroke-width of the circle arc.
fill string 'none' SVG fill colour of the circle interior.
animationDuration string '2s' Duration of one rotation/dash cycle. Accepts any CSS <time>.
size 'sm' | 'md' | 'lg' 'md' Size token controlling the overall diameter.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to global theme when null.
styleClass string | null null Extra CSS class(es) applied to the host element.
ariaLabel string 'Loading...' Screen-reader accessible label.
