StyleClass

A directive that manages CSS class enter/leave transitions on a target element. Useful for animated show/hide panels, menus, and overlays — all driven by CSS, zero JavaScript animation libraries required.

Directive ui-lib-custom/style-class

Toggle Mode

Set toggleClass to add and remove a single CSS class on each click. This is the simplest usage — no transitions, just a class toggle.

This panel is shown/hidden via a CSS class toggle.

No JavaScript animation — just CSS.

<button
  [uiLibStyleClass]="'@next'"
  toggleClass="sc-panel--open"
>
  Toggle Panel
</button>
<div class="sc-panel">...</div>

Fade Animation

Use enterActiveClass / leaveActiveClass with CSS keyframe animations for smooth enter and leave transitions.

Fade Panel

Click outside to close this panel.

<button
  [uiLibStyleClass]="'@next'"
  enterFromClass="hidden"
  enterActiveClass="fade-in"
  leaveActiveClass="fade-out"
  leaveToClass="hidden"
  [hideOnOutsideClick]="true"
>Show Panel</button>
<div class="hidden">...</div>

Slide Animation

Use the full enterDoneClass / leaveDoneClass lifecycle to control the final state independently from the active animation class.

Slide Panel

This panel slides in/out using enterDoneClass and leaveDoneClass.

<button
  [uiLibStyleClass]="'@next'"
  enterFromClass="slide-hidden"
  enterActiveClass="slide-down"
  enterDoneClass="slide-open"
  leaveActiveClass="slide-up"
  leaveDoneClass="slide-hidden"
>Toggle Slide</button>

Special Selectors

Use @parent, @prev, and @grandparent to target elements relative to the trigger without needing CSS IDs.

@prev target
@parent wrapper 
<!-- @prev -->
<div class="target">Prev target</div>
<button [uiLibStyleClass]="'@prev'"
        toggleClass="is-active">Toggle</button>

<!-- @parent -->
<div class="parent">
  <button [uiLibStyleClass]="'@parent'"
          toggleClass="is-active">Toggle</button>
</div>

CSS Selector

Pass any CSS selector to target a specific element elsewhere in the page.

This panel is targeted by #sc-remote-panel CSS selector.

<button
  [uiLibStyleClass]="'#my-panel'"
  toggleClass="is-open"
>Toggle</button>

<div id="my-panel">...</div>

API

Inputs

Input Type Default Description
uiLibStyleClass string required Target selector: @next, @prev, @parent, @grandparent, or a CSS selector.
toggleClass string '' Single class to toggle. When set, bypasses the full enter/leave lifecycle.
enterFromClass string '' Class(es) applied at the start of enter.
enterActiveClass string '' Class(es) applied during enter (e.g. animation class).
enterToClass string '' Class(es) applied at end of enter transition.
enterDoneClass string '' Class(es) kept after enter completes.
leaveFromClass string '' Class(es) applied at the start of leave.
leaveActiveClass string '' Class(es) applied during leave (e.g. animation class).
leaveToClass string '' Class(es) applied at end of leave transition.
leaveDoneClass string '' Class(es) kept after leave completes.
hideOnOutsideClick boolean false When true, clicking outside the target triggers leave / toggles off.
