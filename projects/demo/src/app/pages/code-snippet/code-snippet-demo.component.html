Code Snippet

A styled code display card with macOS-style window chrome, a language-labelled tab bar, optional line numbers, and a one-click copy-to-clipboard button. Supports three design variants (material, bootstrap, minimal), three sizes, and every token is overridable via CSS custom properties.

Basic Usage

Provide the code string and set a language to label the tab. The component renders line numbers and a copy button by default.

With a filename

Setting filename overrides the tab label with the actual file name.

Languages

The language input accepts 'javascript', 'typescript', 'html', 'css', 'scss', 'json', 'bash', 'shell', and 'text'. It controls the tab label and the data-language attribute on the <pre> element, which external highlighters can target.

Design Variants

Three visual variants mirror the global library theme: material (rounded corners, hover lift), bootstrap (sharp corners, prominent border), and minimal (no shadow, transparent border until hovered). When not set, the active ThemeConfigService variant is used automatically.

@for (variant of variants; track variant) {

{{ variant }}

}

Options

Without line numbers

Set [showLineNumbers]="false" for a more compact presentation.

Without window chrome

Set [showWindowChrome]="false" to hide the macOS traffic-light dots.

Minimal chrome

Combine [showWindowChrome]="false" and [showLineNumbers]="false" for the cleanest inline presentation.

Sizes

The size input scales the font and padding: 'sm', 'md' (default), 'lg'.

@for (size of sizes; track size) { }

Max height (scrollable)

Set maxHeight to cap the code body height. The body becomes scrollable when content overflows.

API

Inputs

Input Type Default Description
code string '' Raw code string to display.
language CodeSnippetLanguage 'text' Programming language; drives the tab label and data-language attribute.
filename string | null null Filename shown in the tab bar. Falls back to the language label when null.
variant CodeSnippetVariant | null null Design variant. Inherits ThemeConfigService variant when null.
size 'sm' | 'md' | 'lg' 'md' Controls font size and padding.
showWindowChrome boolean true Show or hide the macOS traffic-light dots.
showLineNumbers boolean true Show or hide the line number gutter.
showCopyButton boolean true Show or hide the copy-to-clipboard button.
maxHeight string | null null CSS max-height on the code body (e.g. '300px'). Body scrolls on overflow.
styleClass string | null null Extra CSS classes appended to the root element.

Outputs

Output Type Description
codeCopied void Emitted after the code is copied to the clipboard.

Accessibility

  • The card root has role="region" with an auto-generated aria-label derived from the filename or language.
  • Line numbers and window chrome are aria-hidden="true" — they are purely decorative and do not interfere with screen reader output.
  • The copy button has a descriptive aria-label ("Copy code to clipboard") and toggles to "Copied!" with aria-pressed="true" after success.
  • The copy button is keyboard-focusable with a visible :focus-visible ring.
  • prefers-reduced-motion disables all transitions and hover transforms.
  • High-contrast mode (forced-colors: active) draws a solid border and preserves dot colours with forced-color-adjust: none.
