A lightweight directive that programmatically focuses the host element after view initialization — a reliable, Angular-first alternative to the native autofocus attribute.

Directive Misc A11y

Add uiLibAutoFocus to any focusable element. Focus is applied automatically after the view initializes.

Bind the autofocus input to control whether focus is applied. Useful when rendering inputs inside dialogs, drawers, or dynamic forms.

@if (showConditional()) { Hide input } @else { Show input } @if (showConditional()) {
}

Set [disabled]="true" to prevent focus from being applied — for example when the feature is toggled off by the user.

Focus enabled: {{ enableFocus() }}

Directive selector

Inputs

Name Type Default Description
disabled boolean false When true, focus is skipped on mount.
selector string | null null Optional child selector used as the focus target instead of the host.
