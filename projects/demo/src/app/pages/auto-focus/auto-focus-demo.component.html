AutoFocus

A lightweight directive that programmatically focuses the host element after view initialization — a reliable, Angular-first alternative to the native autofocus attribute.

Directive Misc A11y

Basic Usage

Add uiLibAutoFocus to any focusable element. Focus is applied automatically after the view initializes. 

<input uiLibAutoFocus />

Conditional Focus

Bind the autofocus input to control whether focus is applied. Useful when rendering inputs inside dialogs, drawers, or dynamic forms.

@if (showConditional()) {
} 
@if (show) {
  <input uiLibAutoFocus />
}

Disable Autofocus

Set [autofocus]="false" to prevent focus from being applied — for example when the feature is toggled off by the user. 

<input uiLibAutoFocus [autofocus]="isEnabled" />

API

Directive selector

import {{ '{' }} AutoFocus {{ '}' }} from 'ui-lib-custom/auto-focus';
[uiLibAutoFocus]

Inputs

Name Type Default Description
autofocus boolean true When true, the host element is focused after view initialization. Set to false to skip focus.
