Basic Usage

Scroll down this page to see the button appear in the bottom-right corner. The default threshold is 400px and the default target is the window.

⬇ Scroll down the page to reveal the ScrollTop button.

<ui-lib-scroll-top />

Custom Threshold

Control how far the user must scroll before the button appears using the threshold input (pixels, default: 400).

Parent Container Target

Set target="parent" to watch a scrollable parent element instead of the window. The button is positioned absolute inside the container. The parent must have position: relative and overflow-y: auto.

Variants

Three design variants: material (default — pill, primary colour + shadow), bootstrap (square corners, Bootstrap blue), and minimal (muted surface + border).

Three sizes are available: sm, md (default), and lg.

Custom Icon

Use the icon input to supply any PrimeIcons class string. The default is pi pi-arrow-up.

API

Inputs

Name Type Default Description
threshold number 400 Scroll distance (px) before the button becomes visible.
target 'window' | 'parent' 'window' Scroll target: the global window or the immediate parent element.
icon string 'pi pi-arrow-up' CSS class(es) for the icon inside the button.
behavior 'smooth' | 'auto' 'smooth' Native scroll-behavior applied when scrolling to top.
buttonAriaLabel string 'Scroll to top' Accessible label for the button element.
size 'sm' | 'md' | 'lg' 'md' Button size.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Falls back to global ThemeConfigService when null.
styleClass string | null null Additional CSS classes added to the host element.

Public properties

Name Type Description
