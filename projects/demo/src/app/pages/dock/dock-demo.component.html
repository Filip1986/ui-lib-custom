Dock

A macOS-style icon bar with a hover magnification effect. Items scale up smoothly when hovered, with neighbouring items receiving a proportional cascade. Supports four positions (bottom, top, left, right), three design variants, and three size tokens.

Basic

Default dock positioned at the bottom with the material variant and magnification enabled. Hover over items to see the magnification effect; click to trigger the command. 

{} osItems: DockItem[] = [
  { label: 'Finder', icon: 'pi pi-folder', command: handler },
  { label: 'Terminal', icon: 'pi pi-code', command: handler },
  // ...
];

<ui-lib-dock [items]="osItems" position="bottom" variant="material" />

Variants

Three design variants are available: material (frosted glass), bootstrap (solid background), and minimal (no background or border).

@for (v of variants; track v) {
{{ v }}
} 
<ui-lib-dock [items]="items" variant="material" />
<ui-lib-dock [items]="items" variant="bootstrap" />
<ui-lib-dock [items]="items" variant="minimal" />

Sizes

The size input controls icon and item dimensions via design tokens.

@for (s of sizes; track s) {
{{ s }}
} 
<ui-lib-dock [items]="items" size="sm" />
<ui-lib-dock [items]="items" size="md" />
<ui-lib-dock [items]="items" size="lg" />

Positions

The dock can be anchored to any edge of its container. For vertical positions (left, right), the list is rendered as a column.

@for (p of positions; track p) {
{{ p }}
} 
<ui-lib-dock [items]="items" position="bottom" />
<ui-lib-dock [items]="items" position="top" />
<ui-lib-dock [items]="items" position="left" />
<ui-lib-dock [items]="items" position="right" />

No Magnification

Set [magnification]="false" to render a static dock without the scale animation — useful in compact layouts or for users who prefer reduced motion. 

<ui-lib-dock [items]="items" [magnification]="false" />

Disabled Items

Individual items can be disabled by setting disabled: true. Disabled items are styled with reduced opacity, are not interactive, and are announced to screen readers via aria-disabled. 

const items: DockItem[] = [
  { label: 'Home', icon: 'pi pi-home', command: handler },
  { label: 'Search', icon: 'pi pi-search', disabled: true },
  // ...
];

Link Items

Items with a url render as <a href> anchors. Items with routerLink use Angular's router. Items with only a command render as <button> elements. 

// Anchor with external URL
{ label: 'Anthropic', icon: 'pi pi-external-link', url: 'https://anthropic.com', target: '_blank' }

// Angular RouterLink
{ label: 'Home', icon: 'pi pi-home', routerLink: '/' }

// Button with command
{ label: 'Settings', icon: 'pi pi-cog', command: handler }

API

Dock Inputs

Input Type Default Description
items DockItem[] [] Array of items to display in the dock.
position 'bottom' | 'top' | 'left' | 'right' 'bottom' Edge of the container where the dock is anchored.
variant 'material' | 'bootstrap' | 'minimal' | null null Design-system variant. Falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Controls item and icon dimensions via design tokens.
magnification boolean true Whether the hover magnification effect is enabled.
magnificationLevel number 1.5 Maximum scale factor applied to the directly hovered item. Values 1.2–2.5 work well.
styleClass string | null null Extra CSS class appended to the host element.

Dock Outputs

Output Type Description
itemClick DockItemCommandEvent Emitted when any enabled item is clicked or activated via keyboard.

DockItem Interface

Property Type Description
label string Accessible label and tooltip text shown on hover.
icon string Icon class(es) applied to the icon element (e.g. 'pi pi-home').
disabled boolean When true, the item is non-interactive and styled as disabled.
visible boolean When explicitly false, the item is excluded from the rendered list.
command (event: DockItemCommandEvent) => void Callback invoked when the item is activated. Renders as a button.
url string External URL. Renders as an <a href> element.
routerLink string | string[] Angular Router link. Renders as a [routerLink] anchor.
target string Anchor target attribute (e.g. '_blank').
styleClass string Extra CSS class added to the item list element.
