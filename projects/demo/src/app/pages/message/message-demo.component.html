API Reference

Inputs

Input Type Default Description
severity 'success' | 'info' | 'warn' | 'error' | 'secondary' | 'contrast' 'info' Controls the colour palette and default icon.
variant 'material' | 'bootstrap' | 'minimal' | null null (falls back to global theme) Design variant. When null, inherits the ThemeConfigService variant.
size 'sm' | 'md' | 'lg' 'md' Size of the message.
text string | null null Optional text content. Can be combined with content projection.
icon string | null null (uses severity default) Custom icon name to override the severity-driven icon.
closable boolean false Shows a close button when true.
styleClass string | null null Additional CSS class(es) appended to the host element.

Outputs

Output Payload Description
close void Emitted when the close button is clicked.
