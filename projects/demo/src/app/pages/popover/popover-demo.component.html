Declarative [(visible)]

Control visibility with a two-way model binding instead of template refs. The popover will position itself relative to the viewport center when no target is set.

{{ declarativeVisible() ? 'Hide' : 'Show' }} Popover

Driven by [(visible)]. Current state: {{ declarativeVisible() ? 'open' : 'closed' }}

API

Inputs

Input Type Default Description
visible boolean false Two-way visibility binding via [(visible)].
header string | null null Optional header text shown at the top of the panel.
showCloseButton boolean false Renders a close (×) button in the header area.
dismissable boolean true Clicking outside the panel closes the popover.
closeOnEscape boolean true Pressing Escape closes the popover.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant; inherits from the global theme when null.
styleClass string | null null Extra CSS classes on the host element.

Outputs

Output Payload Description
shown void Emitted after the popover becomes visible.
hidden void Emitted after the popover is hidden.

Public Methods

Method Description
show(target: HTMLElement) Show the popover anchored to target.
hide() Hide the popover.
toggle(target: HTMLElement) Show if hidden, hide if visible.
