ProgressBar

A horizontal bar that visualises a completion percentage or signals an ongoing operation with a smooth animation.

Basic

Pass a number from 0 to 100 via [value]. The percentage label is shown inside the fill by default.

No label

Set [showValue]="false" to hide the percentage text — useful for thin bars where the label would not fit.

@for (size of sizes; track size) {

size="{{ size }}"

}

Custom label

Override the default percentage text with any string via [label].

Indeterminate

Use mode="indeterminate" when the duration is unknown. The fill bar oscillates continuously. aria-busy="true" and aria-label="Loading" are set automatically for screen readers.

@for (variant of variants; track variant) {

variant="{{ variant }}"

}

Variants

Three design variants — material (pill-shaped, subtle inset shadow), bootstrap (rounded corners, square fill), and minimal (flat, 2 px radius).

@for (variant of variants; track variant) {

variant="{{ variant }}"

}

Sizes

Three size tokens control bar height: sm, md (default), and lg.

@for (size of sizes; track size) {

size="{{ size }}"

}

Custom colour

Supply any valid CSS colour string via [color] to override --uilib-progress-bar-fill-bg for one-off uses.

Dynamic value

The fill transitions smoothly when the value changes reactively.

+10 Reset {{ dynamicValue() }}%

API

Input Type Default Description
value number 0 Progress value from 0 to 100 (clamped automatically).
mode 'determinate' | 'indeterminate' 'determinate' Display mode.
showValue boolean true Whether to show the percentage label inside the fill.
label string | null null Custom label; overrides the computed percentage string.
size 'sm' | 'md' | 'lg' 'md' Bar height size token.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant; inherits from ThemeConfigService when null.
color string | null null Custom fill colour; overrides the CSS variable.
styleClass string | null null Additional CSS classes added to the host element.
