Fluid

A layout wrapper that stretches all descendant form controls — inputs, textareas, selects, and buttons — to 100 % of the available width. Available as both a component (<ui-lib-fluid>) and an attribute directive ([uiLibFluid]) for zero-wrapper usage.

Component Directive Layout Misc

Using the Component

Wrap any form or container with <ui-lib-fluid>. All child inputs, textareas, and buttons grow to fill the full width.

Using the Directive

Apply uiLibFluid as an attribute on any existing element to avoid adding an extra wrapper to the DOM. The directive adds the same ui-lib-fluid CSS class to the host.

Conditional Fluid

Bind [uiLibFluid]="expression" to toggle the fluid layout at runtime. When false, inputs and buttons revert to their natural intrinsic sizes.

Fluid: {{ isFluid() ? 'ON' : 'OFF' }}

Attribute Shorthand

The directive supports attribute-only usage — just add uiLibFluid without a binding. It defaults to true through Angular's booleanAttribute transform.
