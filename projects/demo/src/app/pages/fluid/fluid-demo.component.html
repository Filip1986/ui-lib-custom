The fluid directive and component apply no CSS custom properties. Width behaviour is achieved purely via structural CSS rules targeting host elements.

The fluid layout wrapper applies no ARIA attributes. All accessibility semantics come from the wrapped form controls. Ensure every child input and button has an associated <label> and a descriptive name.
