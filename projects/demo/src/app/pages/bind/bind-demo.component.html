Bind

A utility directive that applies a dynamic record of key-value pairs as DOM properties on the host element — the Angular-first equivalent of PrimeNG's pBind.

Directive Misc

Basic Usage

Pass any object literal to [uiLibBind]. Each key is set as a DOM property on the host element. Using id, title, or custom data-* attributes all work out of the box.

Hover me — tooltip set via uiLibBind 
<div [uiLibBind]="{ id: 'my-box', title: 'Tooltip text', tabIndex: 0 }">
  Content
</div>

Boolean & Numeric Properties

uiLibBind uses Renderer2.setProperty under the hood, so boolean and numeric DOM properties work exactly as expected.

tabIndex = 2 (try Tab)
tabIndex = -1 (not reachable by Tab) 
<!-- numeric property -->
<div [uiLibBind]="{ tabIndex: 2 }">Focusable</div>
<!-- boolean property -->
<div [uiLibBind]="{ hidden: true }">Hidden</div>

Selector

Inputs

Name Type Default Description
uiLibBind Record<string, unknown> {} An object whose enumerable own properties are set as DOM properties on the host element. Properties removed between updates are reset to null.
