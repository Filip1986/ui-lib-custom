This is a behavioral directive with no CSS custom properties. All visual styling is controlled by the wrapped component.

Bind is a declarative attribute-binding directive. Use it to reactively drive ARIA attributes from signals — e.g. [uilibBind]="{ 'aria-expanded': isOpen() }".
