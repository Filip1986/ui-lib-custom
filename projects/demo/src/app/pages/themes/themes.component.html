Theme Switcher

Toggle between light and dark themes by setting data-theme="light|dark" on html or any container. CSS variables in ui-lib-custom/src/lib/themes/themes.css drive component colors.

Toggle in Your App

  1. Import themes.css once globally (see Quick Start).
  2. Set data-theme on html or a container, e.g. via a toggle.
  3. All components consume CSS variables; override them per brand if needed.
