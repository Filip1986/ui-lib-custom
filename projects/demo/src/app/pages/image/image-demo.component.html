a. Basic

Provide src and alt to render a simple image.

b. Preview

Set [preview]="true" to enable the lightbox overlay. Hover over the image to see the indicator, then click to open the overlay. Inside the overlay you can zoom in/out and rotate the image, or press Escape to close.

c. Custom Indicator

Replace the default eye-icon indicator with your own content using the #imageIndicator template slot.

🔍

d. Error Fallback

Pass errorSrc to display an alternative image when the primary source fails to load.

e. Sizes

Use the size input (sm, md, lg) to control border-radius scaling and indicator icon sizing.

md
lg

f. Variants

Three design variants are available: material (subtle shadow), bootstrap (visible border, tighter radius), and minimal (no rounding).

g. Custom Error Template

Use the #imageError template slot to render your own placeholder when the image fails to load and no errorSrc is set.

h. Dimensions

Pass width and height to constrain the image element size. These are forwarded directly to the underlying <img> element.

i. Two-Way Binding

Use [(previewVisible)] to control and react to the overlay open state programmatically.
