Pro & Enterprise Roadmap

The library is growing beyond commodity components. The categories below represent the highest-value, most complex components that enterprise Angular teams need but don't want to build themselves — Angular-native, signals-first, and deeply accessible from day one.

Advanced Data Grid

Virtual scroll, column pinning, Excel-like editing, tree tables, server-side ops, and realtime streaming.

Workflow & Visual Builder

Drag-drop pipeline designer, nested query builder (→ SQL / Mongo / Elastic), BPMN editor, form/survey builder.

Dynamic Form Engine

JSON-schema-driven forms, conditional wizards, complex validation, and large reactive-form performance tooling.

Gantt & Scheduler

Resource scheduling, drag scheduling, dependency management, timeline planner, and calendar integration.

AI-Integrated UI Kit

Streaming chat widget, prompt playground, AI diff/review panels, token-usage viewer, AI workflow builder.

Analytics & Charts

TradingView-like charts, org charts, network graphs, heatmaps, and realtime monitoring dashboard widgets.

Rich Text & Document

Notion-like editor, collaborative editing, PDF annotation viewer, diff viewer, and document approval flow.

Developer Experience

Log viewers, feature-flag dashboards, permission/role editors, API explorer, audit trail viewers.

Angular's smaller ecosystem is an advantage here — much premium React tooling never reaches Angular, or arrives as a mediocre wrapper. "Angular-native and deeply polished" is itself the competitive differentiator.

Profiling Notes

Use Angular DevTools to validate change detection and rendering behavior.

  • Profiler - record a short interaction and verify only affected components render.
  • Change Detection - confirm OnPush components skip renders when inputs are unchanged.
  • Signals - verify derived state comes from computed signals (no effect-driven UI updates).
  • Interactions - watch for unexpected renders when opening Select, Tabs, or Accordion.
