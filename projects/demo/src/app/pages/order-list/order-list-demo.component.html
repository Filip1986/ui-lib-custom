Keyboard shortcuts

Key Action
Arrow Down / Up Navigate focus between items (wraps).
Home / End Jump focus to first / last item.
Space / Enter Toggle selection of focused item.
Ctrl+A Select all visible items.
Escape Clear selection.
Alt+↑ / Ctrl+↑ Move selected items up.
Alt+↓ / Ctrl+↓ Move selected items down.
Alt+Home Move selected items to top.
Alt+End Move selected items to bottom.

ARIA

  • List container: role="listbox", aria-multiselectable="true".
  • Each item: role="option", aria-selected.
  • Control buttons carry configurable aria-label inputs.
  • All reorder and drag moves are announced via LiveAnnouncerService.
  • Provide a label with [ariaLabel] or [ariaLabelledBy].

Inputs

Input Type Default Description
value unknown[] [] Ordered list items — two-way via [(value)].
selection unknown[] [] Selected items — two-way via [(selection)].
header string | null null Static header text.
filterBy string | null null Dot-notation filter field. Shows filter input when set.
filterPlaceholder string 'Filter' Filter input placeholder.
filterMatchMode 'contains' | 'startsWith' | 'endsWith' | 'equals' 'contains' Matching strategy.
filterLocale string | undefined undefined BCP 47 locale for comparisons.
dragDrop boolean false Enables native HTML5 drag-and-drop.
disabled boolean false Disables all interaction.
metaKeySelection boolean false Requires Ctrl/Meta to multi-select.
stripedRows boolean false Alternating row background.
controlsPosition 'left' | 'right' | 'top' 'left' Control buttons position.
variant 'material' | 'bootstrap' | 'minimal' | null null Theme variant override.
size 'sm' | 'md' | 'lg' 'md' Size scale.
trackBy string | null null Property path for identity.
ariaLabel string | null null Listbox accessible label.
ariaLabelledBy string | null null ID(s) labelling the listbox.

Outputs

Output Payload Description
reordered OrderListReorderEvent After any reorder (buttons or DnD).

filtered OrderListFilterEvent When filter query changes.
