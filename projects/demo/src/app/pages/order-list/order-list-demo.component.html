OrderList

A reorderable list component with button controls, optional inline filtering, drag-and-drop, custom item templates, and full keyboard accessibility.

1. Basic

Click items to select them, then use the control buttons (or keyboard shortcuts) to reorder.

{{ item.name }} {{ formatPrice(item.price) }}

2. Filter

Set filterBy="name" to render an inline search field. The visible list updates as you type; the underlying array is never mutated.

{{ item.name }} {{ item.category }}

3. Custom Templates

Use uiOrderListHeader, uiOrderListItem, and uiOrderListEmpty directives for fully custom rendering.

Products ({{ templateProducts().length }})
{{ item.category }} {{ item.name }} {{ starsLabel(item.rating) }} {{ formatPrice(item.price) }}
No products match the current filter.

4. Drag & Drop

Enable [dragDrop]="true" to let users reorder by dragging. A drop indicator line shows the insertion point.

⠿ {{ item.name }}

Last reorder: {{ lastReorderEvent() }}

5. Accessibility

Keyboard shortcuts

KeyAction
Arrow Down / UpNavigate focus between items (wraps).
Home / EndJump focus to first / last item.
Space / EnterToggle selection of focused item.
Ctrl+ASelect all visible items.
EscapeClear selection.
Alt+↑ / Ctrl+↑Move selected items up.
Alt+↓ / Ctrl+↓Move selected items down.
Alt+HomeMove selected items to top.
Alt+EndMove selected items to bottom.

ARIA

  • List container: role="listbox", aria-multiselectable="true".
  • Each item: role="option", aria-selected.
  • Control buttons carry configurable aria-label inputs.
  • All reorder and drag moves are announced via LiveAnnouncerService.
  • Provide a label with [ariaLabel] or [ariaLabelledBy].
