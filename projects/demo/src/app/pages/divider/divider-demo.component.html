Divider

import { Divider } from 'ui-lib-custom/divider'

A visual separator that divides content horizontally or vertically. Supports solid, dashed, and dotted lines, optional projected label content, flexible alignment, and three design variants.

API

Input Type Default Description
orientation 'horizontal' | 'vertical' 'horizontal' Direction of the divider line.
type 'solid' | 'dashed' | 'dotted' 'solid' Visual style of the divider line.
align 'left' | 'center' | 'right' | 'top' | 'bottom' | null null (resolves to 'center') Position of the projected content along the divider. Horizontal dividers support left / center / right; vertical dividers support top / center / bottom.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant; inherits from ThemeConfigService when null.
styleClass string | null null Additional CSS classes added to the host element.
