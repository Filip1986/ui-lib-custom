Segmented progress bar that visualises multiple values as proportional coloured segments. Ideal for storage usage, resource allocation, and survey breakdowns.

Basic

Pass an array of MeterItem objects via [values]. Each item requires a label, a numeric value, and a CSS color. The legend renders below the bar by default.

Label Position

Use labelPosition="start" to move the legend above the bar. Items can include an optional icon (a PrimeIcons class) shown inside the legend swatch.

labelPosition="start"

labelPosition="end" (default)

Sizes

Three bar thicknesses: sm, md (default), and lg.

@for (size of sizes; track size) {

size="{{ size }}"

}

Design Variants

Three design variants: material (pill-shaped, soft shadow), bootstrap (squared corners, no gap), and minimal (flat, almost square). When variant is null (the default), it inherits from ThemeConfigService.

@for (variant of variants; track variant) {

variant="{{ variant }}"

}

Vertical Orientation

Set orientation="vertical" to stack segments vertically. The legend is placed to the right by default (labelPosition="end").

Legend end (right)

Legend start (left)

No Legend

Set [showLabels]="false" to render the bar without a legend — useful as a compact inline indicator.

Inputs

Input Type Default Description
values MeterItem[] [] Array of meter segments to render.
min number 0 Minimum value of the range.
max number 100 Maximum value of the range.
orientation 'horizontal' | 'vertical' 'horizontal' Bar direction.
showLabels boolean true Show or hide the legend.
labelPosition 'start' | 'end' 'end' Legend placement — start is above / left, end is below / right.
size 'sm' | 'md' | 'lg' 'md' Bar thickness size token.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual design variant; falls back to ThemeConfigService when null.
ariaLabel string 'Meter group' Accessible label for the meter container group (aria-label on the group element).
styleClass string | null null Extra CSS classes appended to the host element.

MeterItem

Property Type Required Description
label string Yes Display label shown in the legend and used as the segment aria-label.
value number Yes Numeric value measured against min / max.
color string Yes CSS colour string for the segment fill (hex, rgb, hsl, named colour).
icon string No Optional PrimeIcons class shown inside the legend colour swatch.

CSS Custom Properties

ARIA Attributes

Element Attribute Value Notes
.ui-lib-meter-group__meters role="group" — Groups all meter segments as one semantic meter group.
.ui-lib-meter-group__meters aria-label Value of ariaLabel input Defaults to "Meter group". Provide a meaningful label for the dataset.
.ui-lib-meter-group__meter role="meter" — Announces each segment as a meter to assistive technology.
.ui-lib-meter-group__meter aria-valuenow Segment value (clamped to range) Current value of the segment.
.ui-lib-meter-group__meter aria-valuemin Value of min input Lower bound of the range.
.ui-lib-meter-group__meter aria-valuemax Value of max input Upper bound of the range.
.ui-lib-meter-group__meter aria-label "<label>: <value> of <max>" Human-readable segment description announced by screen readers.
.ui-lib-meter-group__sr-total aria-live="polite" — Hidden live region that announces computed total changes for dynamic data.
.ui-lib-meter-group__sr-total aria-atomic="true" — Reads the total announcement as a complete phrase.

Keyboard Interaction

MeterGroup is informational and non-interactive — it adds no tabbable controls.

Key Behaviour
Tab / Shift+Tab Skips meter segments and legend items — nothing is focusable.
Arrow keys No component-specific keyboard handling.
Enter / Space No component-specific keyboard handling.

Notes

  • Set a meaningful ariaLabel input (e.g. "Storage usage") so screen readers announce the dataset context rather than the generic default.
  • Decorative legend swatches and icons are aria-hidden="true" so screen readers announce only the text label.
  • A hidden aria-live="polite" region announces segment total changes when [values] is updated dynamically.
  • All segment transitions respect prefers-reduced-motion: reduce.
