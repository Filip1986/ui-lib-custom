Segmented progress bar that visualises multiple values as proportional coloured segments. Ideal for storage usage, resource allocation, and survey breakdowns.

Pass an array of MeterItem objects via [values]. Each item requires a label, a numeric value, and a CSS color. The legend renders below the bar by default.

Use labelPosition="start" to move the legend above the bar. Items can include an optional icon (a PrimeIcons class) shown inside the legend swatch.

Three bar thicknesses: sm, md (default), and lg.

Three design variants: material (pill-shaped, soft shadow), bootstrap (squared corners, no gap), and minimal (flat, almost square). When variant is null (the default), it inherits from ThemeConfigService.

MeterGroup is informational and non-interactive — it adds no tabbable controls.

Notes

  • Set a meaningful ariaLabel input (e.g. "Storage usage") so screen readers announce the dataset context rather than the generic default.
  • Decorative legend swatches and icons are aria-hidden="true" so screen readers announce only the text label.
  • A hidden aria-live="polite" region announces segment total changes when [values] is updated dynamically.
  • All segment transitions respect prefers-reduced-motion: reduce.
