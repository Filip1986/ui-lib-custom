MeterGroup

Visualise multiple proportional values as a single segmented bar — ideal for storage, resource usage, and survey breakdowns.

import { MeterGroup } from 'ui-lib-custom/meter-group'

Basic

Pass an array of MeterItem objects via [values]. Each item needs a label, a numeric value, and a CSS color. The legend is rendered below the bar by default. 

storageItems: MeterItem[] = [
  { label: 'Apps',     value: 16, color: '#34d399' },
  { label: 'Messages', value: 8,  color: '#818cf8' },
  { label: 'Media',    value: 24, color: '#fb923c' },
  { label: 'System',   value: 10, color: '#f87171' },
];

<ui-lib-meter-group [values]="storageItems" />

Label position

Use labelPosition="start" to move the legend above (horizontal) or to the left (vertical) of the bar. Icons can be added via the optional icon field on each item (any PrimeIcons class string).

labelPosition="start"

labelPosition="end" (default)

Sizes

Three bar thicknesses: sm, md (default), and lg.

@for (size of ['sm', 'md', 'lg']; track size) {

size="{{ size }}"

}

Variants

Three design variants — material (pill-shaped, soft), bootstrap (squared corners, no gap), and minimal (flat, almost square).

@for (variant of ['material', 'bootstrap', 'minimal']; track variant) {

variant="{{ variant }}"

}

Vertical orientation

Set orientation="vertical" to stack segments vertically. The legend is placed to the right by default (labelPosition="end").

No legend

Set [showLabels]="false" to render the bar without a legend — useful as an inline indicator.

API

MeterGroup inputs

Input Type Default Description
values MeterItem[] [] Array of meter segments to render.
min number 0 Minimum value of the range.
max number 100 Maximum value of the range.
orientation 'horizontal' | 'vertical' 'horizontal' Bar direction.
showLabels boolean true Show or hide the legend.
labelPosition 'start' | 'end' 'end' Legend placement — start is above/left, end is below/right.
size 'sm' | 'md' | 'lg' 'md' Bar thickness size token.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual design variant; falls back to ThemeConfigService when null.
styleClass string | null null Extra CSS classes appended to the host element.

MeterItem

Property Type Required Description
label string Yes Display label shown in the legend and as aria-label.
value number Yes Numeric value measured against min/max.
color string Yes CSS color string for the segment fill.
icon string No Optional PrimeIcons class shown inside the legend swatch.
