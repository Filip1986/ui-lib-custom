a. Basic

A minimal toggle bound with [(checked)]. The default labels are Yes (on) and No (off).

Value: {{ basicChecked() }}

b. Custom Labels

Use onLabel and offLabel to display context-specific text for each state.

c. Icons

Provide onIcon and offIcon to show a different icon for each state. Use iconPos="right" to place the icon after the label.

d. Sizes

Three size tokens are available: sm, md (default), and lg.

e. Variants

All three design variants are supported: material (default rounded, subtle elevation when checked), bootstrap (tighter corners), and minimal (pill-shaped, translucent fills).

@for (variant of variantOptions; track variant) {
{{ variant }}
}

f. Allow Empty

When [allowEmpty]="false" the button cannot be unchecked once activated. Useful for confirm-type toggles.

State: {{ allowEmptyChecked() ? 'confirmed' : 'pending' }}

g. Disabled

Set [disabled]="true" to prevent interaction. Both off and on states are shown at reduced opacity.

h. Forms Integration

ToggleButton implements ControlValueAccessor and works with both [(ngModel)] and reactive forms.
