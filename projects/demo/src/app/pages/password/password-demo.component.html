Basic

Focus the field to reveal the strength panel. Use a <label> with [for]="passRef.passwordId" to correctly associate the label with the auto-generated input id.

Variants

Three design variants: material, bootstrap, and minimal. Unlike most library components, variant defaults to 'material' and does not inherit from ThemeConfigService — set it explicitly.

Sizes

Three sizes: sm, md (default), and lg.

Toggle Mask

Set [toggleMask]="true" to render a show/hide eye icon button. The button uses aria-label and aria-pressed for full screen-reader support.

Clear Button

Set [showClear]="true" to show an inline clear button when the field has a value. Combine with [toggleMask] to show both icons.

Filled Appearance

Set appearance="filled" for a solid-background input instead of the default outlined style.

No Strength Feedback

Set [feedback]="false" to disable the strength meter panel entirely — useful when you need a plain, unadorned password field.

Custom Strength Labels

Override the four strength panel text strings with promptLabel, weakLabel, mediumLabel, and strongLabel.

States

[disabled], [readonly], and [invalid] can be set as inputs or driven by a reactive FormControl.

Reactive Forms

Use [formControl] or formControlName. FormControl.disable() propagates through setDisabledState.

@if (passwordControl.invalid && passwordControl.touched) { @if (passwordControl.hasError('required')) { Password is required. } @else if (passwordControl.hasError('minlength')) { Password must be at least 8 characters. } }

Status: {{ passwordControl.status }} | Value: {{ passwordControl.value ?? '(empty)' }}

Fluid (Full Width)

Set [fluid]="true" to stretch the field to fill its container.

ARIA Attributes

Element Attribute Notes
Native <input> aria-describedby Points to the strength live region (strengthId) when [feedback]="true".
Strength live region aria-live="polite" Announces strength changes: "Password strength: Weak / Medium / Strong / None".
Strength live region aria-atomic="true" Ensures the full announcement is read on each update.
Visual strength meter aria-hidden="true" Decorative — screen readers use the live region only.
Toggle mask button aria-label "Show password" / "Hide password" reflects current state.
Toggle mask button aria-pressed true when password is visible, false when hidden.
Toggle mask / clear icons aria-hidden="true" Decorative SVG icons hidden from assistive technologies.

Notes

  • The component generates unique passwordId and strengthId on each instance. Use #passRef template reference and [for]="passRef.passwordId" on the label.
  • The strength live region is always present in the DOM when [feedback]="true", but visually hidden. Screen readers announce changes without any extra setup.
  • All transitions respect prefers-reduced-motion: reduce.
