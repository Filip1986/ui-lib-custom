Menu

A flexible navigation panel that supports both static (inline) and popup modes. Items can be organised into labelled groups, separated by dividers, and decorated with icons. Popup menus can be triggered by any element via toggle(event).

Basic

In static mode the panel is always visible. Add items via the model input.

With Separator

Add { separator: true } to insert a horizontal divider between groups of related items.

Disabled Items

Set disabled: true on any item to make it non-interactive and visually muted. Disabled items receive aria-disabled="true" and are excluded from keyboard navigation.

Grouped Items

Provide an items array on a top-level entry to create a labelled group. The parent's label becomes the group header.

Popup Mode

Set [popup]="true" and call menu.toggle($event) on the trigger element. The panel floats near the trigger and closes on outside click or Escape.

Popup with Grouped Items

Grouped items work in popup mode too — the groups are rendered inline within the floating panel.

Command Callback & itemClick Output

Use the command callback on individual items or listen to the itemClick output on the component to react to activations.

URL Items

Set url on an item to render it as a native <a> link. Combine with target="_blank" to open in a new tab.

API Reference

Menu Inputs

Name Type Default Description
model MenuItem[] [] Items to render. Top-level items with items act as group headers.
popup boolean false When true, the panel is hidden and shown as a floating popup via toggle().
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Controls item padding and font size.
styleClass string | null null Extra CSS class applied to the host element.
ariaLabel string 'Menu' Accessible label for the menu panel.

Menu Outputs

Name Payload Description
itemClick MenuItemCommandEvent Emitted when a non-disabled leaf item is activated (click or keyboard).
menuShow MouseEvent Emitted when the popup panel becomes visible.
menuHide void Emitted when the popup panel is hidden.

Menu Methods

Name Signature Description
toggle (event: MouseEvent): void Toggles the popup open/closed. No-op in static mode.
show (event: MouseEvent): void Shows the popup anchored to the trigger element. No-op in static mode.
hide (): void Hides the popup. No-op in static mode or when already hidden.

MenuItem Interface

Property Type Description
label string Display text for the item or group header.
icon string Icon class rendered before the label.
disabled boolean When true, the item is non-interactive.
separator boolean When true, renders a horizontal divider line.
visible boolean When false, the item is hidden from the list.
items MenuItem[] Child items. Makes the parent a group header.
url string When set, the item renders as a native anchor link.
target string Anchor target attribute for url items.
styleClass string Extra CSS class added to the item element.
command (event: MenuItemCommandEvent) => void Callback invoked when the item is activated.

Keyboard Navigation

Key Action
↑ / ↓Move focus to previous / next item.
HomeMove focus to the first item.
EndMove focus to the last item.
Enter / SpaceActivate the focused item.
EscapeClose the popup menu (popup mode only).
TabMove focus to the next focusable element outside the menu.
