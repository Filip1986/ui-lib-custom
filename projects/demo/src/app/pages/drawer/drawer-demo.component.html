Drawer

import { Drawer } from 'ui-lib-custom/drawer'

A panel that slides in from the edge of the viewport. Supports four positions, three design variants, and flexible content projection.

Basic

Click to open a right-side drawer with a header and content.

Open Drawer

This is the default drawer sliding in from the right. Use [(visible)] to control open state.

You can project any content inside the drawer body.

API Reference

Inputs

InputTypeDefaultDescription
visiblemodel<boolean>falseTwo-way binding to control open/close.
headerstring''Header title text.
position'left'|'right'|'top'|'bottom''right'Which edge the drawer slides from.
sizestring'300px'Panel width (left/right) or height (top/bottom).
modalbooleantrueShow the semi-transparent backdrop.
closeOnBackdropbooleantrueClose on backdrop click.
closeOnEscapebooleantrueClose on Escape key.
blockScrollbooleantrueLock body scroll while open.
showCloseButtonbooleantrueShow the built-in X close button.
variantDrawerVariant | nullnullVisual variant (inherits from theme service).
styleClassstring | nullnullExtra CSS classes on the host.

Outputs

OutputTypeDescription
shownvoidEmits after the drawer opens.
hiddenvoidEmits after the drawer closes.

SlotSelectorDescription
