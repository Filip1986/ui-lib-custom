A compact label that communicates status, category, or classification at a glance. Supports seven severity colours, three sizes, three design variants, optional icons, and an accessible dismiss button.

Use the severity input to communicate meaning through colour. Seven values are available: primary, secondary, success, info, warn, danger, and contrast.

@for (severity of severities; track severity) { }

Three sizes are available: sm, md (default), and lg.

Set [rounded]="true" to apply fully rounded (pill) corners.

@for (severity of severities; track severity) { }

Provide a PrimeIcons class via the icon input to render a leading icon. Icons are always marked aria-hidden="true" — they are decorative.

Set [dismissible]="true" to render a remove button. The button receives an auto-generated aria-label such as Remove Python tag. The tag does not auto-hide — control visibility from the parent via the (removed) output.

Three design variants: material, bootstrap, and minimal. When variant is null (the default), the active theme from ThemeConfigService is used.

@for (variant of variants; track variant) {

Notes

  • All interactive dismiss controls receive :focus-visible ring styles.
  • Decorative icons — both the leading icon and the dismiss icon — are always marked aria-hidden="true".
  • Transitions are suppressed under @media (prefers-reduced-motion: reduce).
  • Do not rely on colour alone to convey severity — pair it with a visible text label or icon.
