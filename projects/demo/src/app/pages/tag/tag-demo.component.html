Tag

A compact label that communicates status, category, or classification at a glance. Supports seven severity colours, three sizes, three design variants, and optional icons.

Severity

Use the severity input to communicate meaning through colour.

@for (severity of severities; track severity) { }

Sizes

Three sizes are available: sm, md (default), and lg.

Rounded

Set [rounded]="true" to apply fully rounded (pill) corners.

@for (severity of severities; track severity) { }

With Icons

Provide a PrimeIcons class via the icon input to render a leading icon.

Design Variants

Switch between material, bootstrap, and minimal styles.

@for (variant of variants; track variant) {

{{ variant }}

API

Input Type Default Description
value string | null null Text label rendered inside the tag.
icon string | null null PrimeIcons CSS class for a leading icon.
severity 'primary' | 'secondary' | 'success' | 'info' | 'warn' | 'danger' | 'contrast' 'primary' Colour palette for the tag.
rounded boolean false When true, applies fully rounded (pill) corners.
size 'sm' | 'md' | 'lg' 'md' Size of the tag.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant; inherits from ThemeConfigService when null.
styleClass string | null null Additional CSS classes added to the host element.
