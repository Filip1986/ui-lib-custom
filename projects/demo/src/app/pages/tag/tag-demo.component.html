Tag

import { Tag } from 'ui-lib-custom/tag'

A compact label that communicates status, category, or classification at a glance. Supports seven severity colours, three sizes, three design variants, optional icons, and an accessible dismiss button.

Severity

Use the severity input to communicate meaning through colour. Seven values are available: primary, secondary, success, info, warn, danger, and contrast.

@for (severity of severities; track severity) { }

Sizes

Three sizes are available: sm, md (default), and lg.

Rounded

Set [rounded]="true" to apply fully rounded (pill) corners.

@for (severity of severities; track severity) { }

With Icons

Provide a PrimeIcons class via the icon input to render a leading icon. Icons are always marked aria-hidden="true" — they are decorative.

Dismissible

Set [dismissible]="true" to render a remove button. The button receives an auto-generated aria-label such as Remove Python tag. The tag does not auto-hide — control visibility from the parent via the (removed) output.

Design Variants

Three design variants: material, bootstrap, and minimal. When variant is null (the default), the active theme from ThemeConfigService is used.

@for (variant of variants; track variant) {

API

Inputs

InputTypeDefaultDescription
value string | null null Text label rendered inside the tag.
icon string | null null PrimeIcons CSS class for a leading decorative icon.
severity 'primary' | 'secondary' | 'success' | 'info' | 'warn' | 'danger' | 'contrast' 'primary' Colour palette for the tag.
rounded boolean false Applies fully rounded (pill) corners.
size 'sm' | 'md' | 'lg' 'md' Size of the tag.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant; inherits from ThemeConfigService when null.
styleClass string | null null Additional CSS classes added to the host element.
dismissible boolean false Renders an inline dismiss button.
removeIcon string 'pi pi-times' CSS class for the dismiss button icon.

Outputs

OutputTypeDescription
removed OutputEmitterRef<MouseEvent> Emitted when the dismiss button is clicked. The tag is not auto-removed — control visibility from the parent.

Accessibility

ARIA Attributes

AttributeElementValueNotes
id Host ui-lib-tag-{n} Auto-generated, unique per instance.
role Host status or group status by default; switches to group when dismissible is true so AT announces both the label and the button.
aria-label Host value Set to the tag's text value. Omitted when value is null.
aria-hidden Leading icon true Decorative icon is always hidden from assistive technology.
aria-label Dismiss button Remove {value} tag Auto-generated from value. Falls back to Remove tag.
aria-hidden Dismiss button icon true Decorative button icon is hidden from assistive technology.

Keyboard Interaction

KeyTargetAction
Tab Dismiss button Moves focus to the button when dismissible is true.
Enter / Space Dismiss button Activates the native button and emits the removed event.

Notes

  • All interactive dismiss controls receive :focus-visible ring styles.
  • Decorative icons — both the leading icon and the dismiss icon — are always marked aria-hidden="true".
  • Transitions are suppressed under @media (prefers-reduced-motion: reduce).
  • Do not rely on colour alone to convey severity — pair it with a visible text label or icon.
