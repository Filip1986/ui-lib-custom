Slider

A linear track control for selecting a numeric value or range. Supports drag interaction, click-to-jump, keyboard navigation, single and range modes, horizontal/vertical orientations, ngModel and reactive forms integration, three size tokens, three design variants, and full CSS-variable theming.

a. Basic

Default horizontal slider with ngModel binding. Drag the handle or use arrow keys.

Value: {{ basicValue }}

b. Range

Enable [range]="true" to get a two-handle range picker. The value becomes a [start, end] tuple.

Range: {{ rangeValue[0] }} – {{ rangeValue[1] }}

c. Step

Use [step] to control snap granularity. This slider snaps to 0, 25, 50, 75, 100.

step=25 · value={{ stepValue }}

d. Min / Max

Constrain the value range. This slider spans −50 to +50 in steps of 10.

value={{ minmaxValue }}

e. Vertical

Set orientation="vertical" to render a vertical track. Drag up to increase.

value={{ verticalValue }}

f. Sizes

Three size tokens: sm, md (default), and lg.

sm
md
lg

g. Animated

Pass [animate]="true" to enable smooth CSS transitions on the fill bar and handle.

value={{ animateValue }}

h. Disabled

A disabled slider is visually muted and ignores all interaction.

disabled

i. Read-only

A read-only slider is focusable but does not respond to drag or keyboard input.

readonly

j. Reactive Forms

Sliders work with Angular reactive forms out of the box via CVA.
