OrganizationChart

A hierarchical tree chart for visualising organisational structures. Supports collapsible subtrees, single and multiple node selection, custom node templates, and three design variants.

1. Basic

Pass an array of OrganizationChartNode objects to [value]. Set expanded: true on any node to show its children by default.

2. Collapsible

Add [collapsible]="true" to display a toggle button on every node that has children. Clicking it fires (nodeExpand) or (nodeCollapse).

3. Single Selection

Set selectionMode="single" and bind [(selection)] to track the selected node. Clicking a selected node deselects it.

Selected: {{ formatSelection(singleSelection()) }}

4. Multiple Selection

Set selectionMode="multiple" to allow selecting several nodes simultaneously. The selection binding holds an array.

Selected: {{ formatSelection(multipleSelection()) }}

5. Custom Template

Project an <ng-template uiOrgChartNode let-node> to replace the default label rendering. The context variable exposes the full OrganizationChartNode including any data you attach.

{{ initials(node.label) }} {{ node.label }} {{ node.data?.role }}

6. Bootstrap Variant

Set variant="bootstrap" for a flat, border-based style consistent with Bootstrap-themed applications.

7. Minimal Variant

variant="minimal" removes all card chrome, using only dashed connector lines for a lightweight look.

8. Accessibility

  • Each node is a button fully keyboard-focusable and activatable.
  • Collapsed subtrees are hidden with aria-hidden="true".
  • Expand/collapse state is announced via LiveAnnouncerService.
  • Selection uses aria-selected on node buttons.
  • Provide a label with [ariaLabel] on the component.
