Tree

Hierarchical data display with expand/collapse, selection modes, filtering, custom templates, and three design variants.

Basic Tree

A read-only tree with no selection. Click the chevron to expand or collapse subtrees.

Single Selection

Click a node to select it. Click the same node again to deselect.

Selected: {{ getSelectionLabel(singleSelection(), 'single') }}

Multiple Selection

Click any number of nodes to build up a selection. Click again to deselect.

Selected: {{ getSelectionLabel(multipleSelection(), 'multiple') }}

Checkbox Selection

Checking a parent cascades to all its descendants. A dash indicates a partially selected parent.

Selected: {{ getSelectionLabel(checkboxSelection(), 'multiple') }}

Filter

Type into the search box to filter the tree. Lenient mode (default) shows matching nodes and all their ancestors.

Custom Node Template

Use ng-template uiTreeNode with an optional uiTreeNodeType to override rendering per node type. Unmatched nodes fall back to the default template.
