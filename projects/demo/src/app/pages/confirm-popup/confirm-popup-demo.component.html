The accept and reject buttons support six severity values: primary, secondary, success, danger, warning, info.

@for (severity of severities; track severity) { {{ severity | titlecase }} }

The popup supports three design variants: material (pill buttons, large radius), bootstrap (border, square corners), and minimal (subtle, flat). Each trigger below opens a popup in its respective variant.

@for (variant of variants; track variant) { {{ variant | titlecase }} }
