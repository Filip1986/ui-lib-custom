a. Basic

Default knob with ngModel binding. Drag up/down or use arrow keys to change the value.

Value: {{ basicValue }}

b. Sizes

Three size tokens: sm, md (default), and lg.

sm
md
lg

c. Step

Use [step] to control the increment. This knob snaps to 0, 25, 50, 75, 100.

step=25 · value={{ stepValue }}

d. Min / Max

Constrain the value range. This knob spans −50 to +50 in steps of 10.

value={{ minmaxValue }}

e. Value Template

Pass a valueTemplate string to format the centre label. Use {value} as the interpolation placeholder.

degrees
percent

f. Custom Colors

Override arc and text colours per-instance via valueColor and textColor inputs.

amber
emerald
red

g. Disabled

A disabled knob is visually muted and ignores all interaction.

disabled

h. Read-only

A read-only knob is focusable but does not respond to drag or keyboard input.

readonly

i. Reactive Forms

Knobs work with Angular reactive forms out of the box via CVA.

brightness={{ brightnessControl.value }}
contrast={{ contrastControl.value }}

API Reference

All inputs accepted by uilib-knob. Import from ui-lib-custom/knob.
