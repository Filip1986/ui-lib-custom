TreeTable

Hierarchical data displayed as an expandable table. Supports sorting, global filter, single / multiple / checkbox selection, custom cell templates, and all three design variants.

Basic — File System

A three-level file-system tree. The name column is the expander column. Click the toggle arrows to expand or collapse folders.

Sorting

Click any sortable column header to cycle: unsorted → ascending → descending → unsorted. Sorting is applied within each group of siblings independently.

Global Filter

Type in the search box to filter rows. Parent nodes remain visible when any of their descendants match the search text.

Selection Modes

Switch between single, multiple, and checkbox selection. Checkbox mode cascades selection to all descendant rows and shows a "partial" state on parents.
