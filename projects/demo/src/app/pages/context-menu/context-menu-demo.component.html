ContextMenu

A floating overlay menu displayed on right-click (or programmatically via show(event)). Supports nested submenus, separators, disabled items, icons, keyboard navigation, and three design variants.

API

Input Type Default Description
model ContextMenuItem[] [] Array of menu items to display.
global boolean false When true, listens to contextmenu events on the entire document automatically.
variant 'material' | 'bootstrap' | 'minimal' | null null Design-system variant; falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Controls font size and item padding.
styleClass string | null null Extra CSS class appended to the host element.
ariaLabel string 'Context Menu' Accessible label applied to the menu panel (aria-label).

Outputs

Output Type Description
itemClick ContextMenuItemCommandEvent Emitted when a non-disabled leaf item is activated.
menuShow MouseEvent Emitted when the menu becomes visible.
menuHide void Emitted when the menu is hidden.

Public Methods

Method Signature Description
show (event: MouseEvent): void Opens the menu at the cursor position. Calls event.preventDefault() to suppress the native context menu.
hide (): void Hides the menu. No-op when already hidden.
toggle (event: MouseEvent): void Toggles visibility at the cursor position.

ContextMenuItem interface

Property Type Description
label string? Display text for the item.
icon string? Icon class name rendered before the label.
disabled boolean? When true, the item is non-interactive.
separator boolean? When true, renders a visual separator instead of a menu item.
visible boolean? When explicitly false, the item is excluded from the rendered list.
styleClass string? Extra CSS class on the item element.
items ContextMenuItem[]? Nested items displayed as a submenu.
command (event: ContextMenuItemCommandEvent) => void Callback invoked when the item is activated.

Keyboard Navigation

Key Behaviour
↓ / ↑ Move focus to the next / previous enabled menu item (skips separators and disabled).
→ Open the submenu of the focused item and move focus to its first item.
← Close the current submenu and return focus to the parent item.
Enter / Space Activate the focused item.
Escape Close the menu (or close just the submenu from within a submenu).
Home / End Jump to the first / last item in the top-level menu.
