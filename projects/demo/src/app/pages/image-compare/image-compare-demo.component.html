Provide leftImage, leftAlt, rightImage, and rightAlt. Drag the handle to reveal more of either image. Use ArrowLeft / ArrowRight keys for keyboard navigation.

Use [(value)] to read and control the slider position (0–100) programmatically.

Position: {{ position() }}%

Use the size input (sm, md, lg) to scale the handle button.

sm
md
lg

Three design variants: material (elevated shadow), bootstrap (border, tighter radius), and minimal (no rounding).

material
bootstrap
minimal

Set [disabled]="true" to prevent the user from moving the slider.

Use the ariaLabel input to provide a descriptive accessible label for the slider handle.

Input Type Default Description
leftImage string '' URL of the left (before) image.
leftAlt string '' Alt text for the left image.
rightImage string '' URL of the right (after) image.
rightAlt string '' Alt text for the right image.
value (model) number 50 Slider position as a percentage (0–100). Supports two-way binding.
disabled boolean false When true, the slider cannot be moved.
variant ImageCompareVariant | null null Design variant; inherits from ThemeConfigService when null.
size ImageCompareSize 'md' Handle size token (sm / md / lg).
styleClass string | null null Extra CSS class on the host element.
ariaLabel string 'Image comparison slider' Accessible label for the slider handle.
Output Type Description
slideStart number Emitted with the current position when the user starts dragging.
slideEnd number Emitted with the final position when the user releases the handle.
Keyboard Description
ArrowRight / ArrowUp Move divider 1% to the right.
ArrowLeft / ArrowDown Move divider 1% to the left.
PageUp Move divider 10% to the right.
PageDown Move divider 10% to the left.
Home Move divider to 0% (far left).
End Move divider to 100% (far right).
