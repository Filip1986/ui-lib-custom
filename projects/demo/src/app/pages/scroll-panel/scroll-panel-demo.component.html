ScrollPanel

A styled scrollable container with variant-aware custom scrollbar theming. Constrain width and height via CSS or inline styles; content that overflows will scroll with beautifully themed scrollbars.

Basic Usage

Wrap any content in <ui-lib-scroll-panel> and constrain its height. The inner content scrolls vertically while the custom scrollbar is styled to match the active variant.

@for (paragraph of longParagraphs; track paragraph) {

{{ paragraph }}

}

Variants

Three design variants are available: material, bootstrap, and minimal. Each variant applies a distinct scrollbar style and border treatment.

}

Horizontal & Both Axes

When the content is wider than the container, a horizontal scrollbar appears. Both axes are supported simultaneously.

{{ item.label }}
}

Interactive Variant Switcher

Switch between variants at runtime. The scrollbar styling updates immediately via CSS custom properties.

Custom Scrollbar via CSS Properties

All scrollbar visuals are driven by CSS custom properties. Override them on the component or a parent to create a bespoke style without changing the variant.

API Reference

Inputs

Name Type Default Description
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Falls back to ThemeConfigService when null.
styleClass string | null null Extra CSS classes applied to the host element.

CSS Custom Properties

Property Description
--uilib-scroll-panel-bg Background colour of the container
--uilib-scroll-panel-border-color Border colour
--uilib-scroll-panel-border-radius Border radius
--uilib-scroll-panel-scrollbar-width Width (and height) of the scrollbar
--uilib-scroll-panel-scrollbar-track-bg Track background colour
--uilib-scroll-panel-scrollbar-thumb-bg Thumb colour (default state)
--uilib-scroll-panel-scrollbar-thumb-bg-hover Thumb colour on hover
--uilib-scroll-panel-scrollbar-radius Thumb and track border radius
--uilib-scroll-panel-transition Transition duration for colour changes
