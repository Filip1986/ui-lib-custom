Chip

Compact element representing an attribute, tag, or dismissible item. Supports an optional icon or image, three sizes, and three design variants. Use the removable input to add a dismiss button that emits an onRemove event.

Component Misc A11y

Basic

Set the label input to display a simple text chip. 

<ui-lib-chip label="Action" />

Icon

Add an icon (a PrimeIcons CSS class) to decorate the chip. 

<ui-lib-chip label="Angular" icon="pi pi-bolt" />

Image

Provide an image URL to render a circular thumbnail. The imageAlt input sets its accessible description. 

<ui-lib-chip label="Amy" image="/assets/amy.png" imageAlt="Amy Elsner" />

Removable

Set [removable]="true" to display a close button. Clicking it emits the (onRemove) event — the chip itself does not disappear automatically; you control visibility in the parent.

Sizes

Three sizes available: sm, md (default), and lg.

@for (size of sizes; track size) { } 
<ui-lib-chip label="sm" size="sm" />
<ui-lib-chip label="md" size="md" />
<ui-lib-chip label="lg" size="lg" />

Variants

Three design variants: material (elevated, pill-shaped), bootstrap (badge-like, square corners), and minimal (muted surface with border).

@for (variant of variants; track variant) {

Options

API

Name Type Default Description
label string | null null Text displayed inside the chip.
icon string | null null CSS class for a leading icon (e.g. "pi pi-user"). Ignored when image is set.
image string | null null URL of a circular thumbnail image to display at the start of the chip.
imageAlt string "Chip" Accessible alt text for the chip image.
removable boolean false When true, renders a close button.
removeIcon string "pi pi-times" CSS class for the remove button icon.
size "sm" | "md" | "lg" "md" Size of the chip.
variant "material" | "bootstrap" | "minimal" | null null Design variant. Inherits from ThemeConfigService when null.
styleClass string | null null Additional CSS classes applied to the host element.
