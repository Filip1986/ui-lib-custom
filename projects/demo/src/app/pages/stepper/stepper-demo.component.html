API Reference

Stepper — ui-lib-stepper

Input Type Default Description
activeStep number 0 Active step index (0-based). Supports two-way binding with [(activeStep)].
linear boolean false When true, users must progress through steps sequentially.
orientation 'horizontal' | 'vertical' 'horizontal' Layout orientation of the stepper.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService when null.
styleClass string | null null Additional CSS classes to apply to the host element.
Output Payload Description
stepChange StepChangeEvent Emitted after the active step changes. Contains activeStep and previousStep.
Method Description
nextStep() Advances to the next step if one exists.
prevStep() Returns to the previous step if one exists.
goToStep(index) Navigates directly to the given step index.
isFirstStep() Returns true when on the first step.
isLastStep() Returns true when on the last step.

StepperPanel — ui-lib-stepper-panel

Input Type Default Description
header string '' Text label shown in the step header.
disabled boolean false Prevents navigation to this step.
Content Slot Template Variable Description
Step content #stepperContent Main body rendered in the active step's content area.
Step footer #stepperFooter Footer area — typically holds navigation buttons.
Custom header #stepperHeader Custom HTML for the step header label (overrides header input).
