Accessibility

SpeedDial follows a menu-button interaction model and exposes keyboard semantics for trigger and menuitem navigation.

  • Trigger exposes aria-haspopup, aria-expanded, and aria-controls.
  • Action list uses role="menu"; actions use role="menuitem".
  • Keyboard support includes Enter/Space toggle, Escape close, arrow navigation, and Home/End focus movement.
  • Disabled actions expose aria-disabled="true" and block command execution.
  • This page logs command output in an aria-live="polite" region at the top.
