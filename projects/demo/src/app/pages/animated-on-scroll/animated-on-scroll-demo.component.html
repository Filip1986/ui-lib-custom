Apply the directive with the uiLibAnimateOnScroll attribute selector. Import preset classes in your styles.scss: @use 'path/to/animate-on-scroll.scss';

Reference
Name Type Default Description
enterClass input<string> '' Space-separated CSS class(es) added when the element enters the viewport.
leaveClass input<string> '' Space-separated CSS class(es) added when the element leaves the viewport (once must be false).
threshold input<number> 0.1 Ratio of element visibility required to trigger (0–1).
rootMargin input<string> '0px' IntersectionObserver root margin (e.g. '-50px 0px').
once input<boolean> true Unobserves the element after the first enter event.
disabled input<boolean> false Skips observer setup entirely when true.
(enter) output<void> — Emitted each time the element enters the viewport.
(leave) output<void> — Emitted each time the element leaves the viewport (once must be false).
--uilib-animate-on-scroll-duration CSS var 600ms Transition duration for all built-in presets.
--uilib-animate-on-scroll-easing CSS var cubic-bezier(0.4,0,0.2,1) Easing function for all built-in presets.
--uilib-animate-on-scroll-delay CSS var 0ms Per-element delay — set inline for stagger effects.
--uilib-animate-on-scroll-distance CSS var 30px Translation distance used by slide-* presets.

AnimateOnScroll respects the user's motion preference. When prefers-reduced-motion: reduce is active, the directive applies the enter class immediately without any animation so content is never hidden or delayed.
