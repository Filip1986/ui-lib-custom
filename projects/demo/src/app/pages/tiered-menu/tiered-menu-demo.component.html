A hierarchical flyout menu supporting arbitrarily deep nesting. Works in both inline and popup modes. Fully accessible with roving focus, ARIA roles, and keyboard navigation.

TieredMenu renders an accessible, keyboard-navigable menu panel that supports arbitrarily deep nesting. Items with an items array open a flyout sub-panel on hover or ArrowRight. Two modes are available:

  • Inline mode (default) — the panel renders in the document flow; ideal for sidebars or always-visible navigation.
  • Popup mode ([popup]="true") — the panel floats as a fixed overlay anchored to the trigger element; control visibility with toggle(), show(), or hide().

The default inline mode renders the menu panel directly in the document flow. Pass a TieredMenuItem[] via [model] and listen for activations with (itemClick).

Inputs

Name Type Default Description

popup boolean false When true, renders as a floating fixed overlay. Control with toggle(), show(), or hide().
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Size token affecting font size, padding, and minimum panel width.
styleClass string | null null Extra CSS class applied to the host element.
ariaLabel string 'Menu' Accessible name for the root <ul role="menu"> panel.

Outputs

Name Payload Description
itemClick TieredMenuItemCommandEvent Emitted when a non-disabled leaf item is activated (click or keyboard).
menuShow MouseEvent | KeyboardEvent Emitted when the popup panel becomes visible. Popup mode only.
menuHide void Emitted when the popup panel is hidden. Popup mode only.

Methods

Name Parameters Description
show(event) MouseEvent | KeyboardEvent Opens the popup anchored to event.currentTarget. No-op in inline mode.
hide() — Closes the popup and restores focus to the trigger element. No-op in inline mode.
toggle(event) MouseEvent | KeyboardEvent Calls show() or hide() based on current visibility.

TieredMenuItem

Property Type Description
label string? Display text for the item.
icon string? Icon class applied to a <span aria-hidden="true">.
items TieredMenuItem[]? Child items — renders a right-arrow caret and opens a flyout sub-panel on hover or ArrowRight.
disabled boolean? When true, item is non-interactive; gets aria-disabled="true" and tabindex="-1".
separator boolean? Renders a <li role="separator"> horizontal divider.
visible boolean? When explicitly false, excludes the item from the rendered list.
url string? Renders the item as <a href="...">.
target string? target attribute for URL-based items (e.g. '_blank').
command function? Callback invoked when the item is activated.
styleClass string? Extra CSS class added to the <li> element.

Notes

  • Focus is restored to the element that triggered the popup when the panel closes via Escape, Tab, or a click outside.
