TieredMenu

A hierarchical flyout menu supporting arbitrarily deep nesting. Works in both inline and popup modes.

Basic (Inline)

The default inline mode renders the menu panel directly in the document flow.

<ui-lib-tiered-menu [model]="items" (itemClick)="onItemClick($event)" />

Nested Submenus

Items with an items array open a flyout sub-panel on hover or keyboard navigation. Nesting is recursive with no depth limit.

const items: TieredMenuItem[] = [

Popup Mode

Set [popup]="true" and control visibility via toggle(event), show(event), or hide().

<button (click)="menu.toggle($event)">Open Menu</button>
<ui-lib-tiered-menu #menu [model]="items" [popup]="true" />

Variants

Choose from material, bootstrap, or minimal.

Sizes

Three sizes available: sm, md (default), and lg.

Item States

Items support disabled, visible: false (hidden), separators, and command callbacks.

URL Items

Set url and optionally target on an item to render it as an anchor tag.

TieredMenu API

Inputs

NameTypeDefaultDescription
modelTieredMenuItem[][]Array of items to display.
popupbooleanfalseWhen true, renders as a floating popup; use toggle/show/hide to control visibility.
variantTieredMenuVariant | nullnullDesign variant: material | bootstrap | minimal. Falls back to ThemeConfigService.
sizeTieredMenuSize'md'Size token: sm | md | lg.
styleClassstring | nullnullExtra CSS class added to the host element.
ariaLabelstring'Menu'Accessible label for the root menu panel.

Outputs

NamePayloadDescription
itemClickTieredMenuItemCommandEventEmitted when a non-disabled leaf item is activated.
menuShowMouseEvent | KeyboardEventEmitted when the popup panel becomes visible (popup mode only).
menuHidevoidEmitted when the popup panel is hidden (popup mode only).

Methods

NameParametersDescription
show(event)MouseEvent | KeyboardEventShows the popup panel anchored to the trigger element. No-op in inline mode.
hide()—Hides the popup panel. No-op in inline mode.
toggle(event)MouseEvent | KeyboardEventToggles the popup panel. No-op in inline mode.

TieredMenuItem Interface

PropertyTypeDescription
labelstring?Display text for the item.
iconstring?Icon class or semantic icon name.
itemsTieredMenuItem[]?Nested children — renders a right-arrow and a flyout sub-panel.
disabledboolean?Prevents interaction.
separatorboolean?Renders a horizontal divider instead of an item.
visibleboolean?When false, hides the item. Defaults to true.
urlstring?Renders the item as an anchor tag.
targetstring?target attribute for url items.
commandfunction?Callback invoked when the item is activated.
styleClassstring?Extra CSS class on the item element.

Keyboard Navigation

KeyAction
ArrowDown / ArrowUpNavigate between items in the same list.
ArrowRightOpen the flyout for an item with nested children.
ArrowLeft / EscapeClose the current flyout and return focus to the parent.
Enter / SpaceActivate the focused item.
Home / EndJump to the first or last item in the current list.
Escape (popup)Close the entire popup menu.
