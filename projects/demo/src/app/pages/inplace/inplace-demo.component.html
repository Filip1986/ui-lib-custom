API

Inputs

Name Type Default Description
active boolean false Two-way bindable; controls whether the editor is shown.
disabled boolean false Prevents activation when set to true.
closable boolean false Renders a close button inside the content slot.
closeIcon string 'pi pi-times' CSS class for the close button icon.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant; falls back to global ThemeConfigService when null.
styleClass string | null null Extra CSS classes added to the host element.

Outputs

Name Type Description
(activated) void Emitted when the component transitions to the active (editing) state.
(deactivated) void Emitted when the component transitions back to the display (inactive) state.

Content Projection

Slot selector Description
[inplaceDisplay] Shown when the component is inactive. Click activates the editor.
[inplaceContent] Shown when the component is active. Any element or component can be projected here.
