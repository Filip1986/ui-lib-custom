A directive that adds a Material-style circular wave effect to any element on click. Apply [uiLibRipple] to any host — button, card, list item, or custom element. Purely decorative: it does not alter ARIA semantics or keyboard behaviour.

Primary Secondary Danger

Basic

Add uiLibRipple to any element. The default ripple colour is a semi-transparent white, suitable for dark or coloured surfaces.

Control the animation speed with [rippleDuration]. Accepts any CSS time value such as '200ms' or '1.2s'.

Disabled

Set [disabled]="true" to suppress the wave while keeping the host element fully interactive. Toggle the button below to compare the two states.

Any Element

The directive works on any HTML element — <div>, <li>, images, or custom components. The host automatically receives position: relative and overflow: hidden.

Inputs

Selector: [uiLibRipple]

Input Type Default Description
disabled boolean false When true, no ripple wave is produced on click.
rippleColor string '' Inline override for --uilib-ripple-color. Any valid CSS colour (e.g. rgba(0,0,0,0.12)).
rippleDuration string '' Inline override for --uilib-ripple-duration (e.g. '400ms').

CSS Custom Properties

Variable Default Description
--uilib-ripple-color rgba(255, 255, 255, 0.35) Wave background colour.
--uilib-ripple-duration 600ms Animation duration.
--uilib-ripple-easing cubic-bezier(0.4, 0, 0.2, 1) Animation timing function.

ARIA Attributes

Host element none added The directive is purely decorative — it does not alter role, aria-*, or tab-stop behaviour.

Notes

  • The Ripple directive is a visual enhancement only — it carries no semantic meaning and does not change the host's ARIA attributes or keyboard behaviour.
  • When the user has enabled Reduce Motion, the ripple is completely suppressed in two layers: the JavaScript handler skips spawning the wave element, and a CSS @media (prefers-reduced-motion: reduce) block sets animation: none; display: none as a defence-in-depth measure.
  • The click listener is only registered in browser environments (isPlatformBrowser check) — SSR-safe.
