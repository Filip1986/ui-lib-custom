Ripple

A directive that adds a Material-style circular wave effect to any element on click. Drop uiLibRipple on any host — button, card, list item, or custom element.

import { Ripple } from 'ui-lib-custom/ripple';

Basic

Add uiLibRipple to any element. The default ripple colour is a semi-transparent white, suitable for dark or coloured surfaces.

Dark ripple (light surfaces)

Use [rippleColor]="'rgba(0,0,0,0.12)'" for light-coloured or white surfaces where white would be invisible.

Click me — card surface

Custom duration

Control speed with [rippleDuration]. Use CSS values like '300ms' or '1s'.

Disabled

Set [disabled]="true" to suppress the ripple while keeping the host interactive. Toggle the button below to see the difference.

Any element

The directive can be applied to any HTML element — <div>, <li>, images, or custom components.

API

Selector

[uiLibRipple]

Inputs

Input Type Default Description
disabled boolean false When true, no ripple wave is produced on click.
rippleColor string '' Inline override for --uilib-ripple-color. Any valid CSS colour (e.g. rgba(0,0,0,0.12)).
rippleDuration string '' Inline override for --uilib-ripple-duration (e.g. 400ms).

CSS variables

Variable Default Description
--uilib-ripple-color rgba(255, 255, 255, 0.35) Wave background colour.
--uilib-ripple-duration 600ms Animation duration.
--uilib-ripple-easing cubic-bezier(0.4, 0, 0.2, 1) Animation timing function.
