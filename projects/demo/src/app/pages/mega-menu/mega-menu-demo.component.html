MegaMenu

A horizontal (or vertical) navigation bar where top-level items open multi-column mega panels of grouped sub-items. Supports three design variants, three sizes, keyboard navigation, disabled items, separators, URL links, and command callbacks.

Basic Horizontal

Default horizontal orientation. Click a top-level item with sub-items to open its mega panel. Simple items (like "Documentation") act as direct links. 

<ui-lib-mega-menu [model]="items" />

Vertical Orientation

Set orientation="vertical" for a stacked list. Mega panels open to the right. 

<ui-lib-mega-menu [model]="items" orientation="vertical" />

Disabled Items & Separators

Top-level items and sub-items can be disabled. Sub-items support separator lines to visually group related entries. 

{ label: "Save As", disabled: true }
{ separator: true }

Command Callbacks

Sub-items can invoke a command function when activated. The last activated item is shown below.

{ label: "Delete", command: ({ item }) => console.log(item) }

API Reference

MegaMenu Inputs

Input Type Default Description
model MegaMenuItem[] [] Array of top-level navigation items.
orientation 'horizontal' | 'vertical' 'horizontal' Layout direction of the navigation bar.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService when null.
size 'sm' | 'md' | 'lg' 'md' Size token applied to link padding and font size.
styleClass string | null null Extra CSS class appended to the host element.
ariaLabel string 'Navigation' Accessible label for the nav landmark.

MegaMenu Outputs

Output Payload Description
itemClick MegaMenuCommandEvent Emitted when a non-disabled sub-item is activated (click or keyboard).

MegaMenuItem Interface

Property Type Description
label string Display label for the top-level item.
icon string Icon CSS class rendered before the label.
items MegaMenuSubColumn[] When present, opens a mega panel with these columns.
disabled boolean Prevents interaction with the top-level item.
visible boolean When false, hides the item entirely.
url string Renders the item as an anchor tag when no sub-items are present.
target string Target attribute for url-based items (e.g. '_blank').
styleClass string Extra CSS class on the root item element.
command (event) => void Callback for simple items without sub-items.

MegaMenuSubColumn Interface

Property Type Description
header string Optional column header label rendered above the item list.
items MegaMenuSubItem[] Sub-items displayed in this column.

MegaMenuSubItem Interface

Property Type Description
label string Display label.
icon string Icon CSS class.
disabled boolean Prevents interaction.
visible boolean When false, hides the item.
separator boolean Renders a visual divider line.
url string Renders as an anchor tag.
target string Target for url-based items.
styleClass string Extra CSS class on the item element.
command (event) => void Callback invoked on activation.
