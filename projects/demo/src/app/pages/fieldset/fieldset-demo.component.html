Fieldset

A labelled content container with an optional collapse/expand animation. Use it to group related form fields or information sections under a clear heading.

import { Fieldset } from 'ui-lib-custom/fieldset';

Basic

A simple non-toggleable fieldset with a legend label and body content. 

<ui-lib-fieldset legend="Personal Information">
  <!-- content -->
</ui-lib-fieldset>

Toggleable

Set [toggleable]="true" to let users collapse and expand the panel. Use [(collapsed)] for two-way binding.

Collapsed: {{ isBasicCollapsed() }} 

<ui-lib-fieldset
  legend="Advanced Options"
  [toggleable]="true"
  [(collapsed)]="isCollapsed"
>
  <!-- content -->
</ui-lib-fieldset>

Pre-collapsed

Pass [collapsed]="true" to start the fieldset in its collapsed state.

This content is hidden initially. Click the legend to reveal it. 

<ui-lib-fieldset
  legend="Hidden by Default"
  [toggleable]="true"
  [collapsed]="true"
>
  <p>Revealed on click</p>
</ui-lib-fieldset>

Custom Legend

Project any HTML into the legend area using the [fieldsetLegend] attribute selector.

User Profile

Custom legend content with icons and formatted text. 

<ui-lib-fieldset [toggleable]="true">
  <span fieldsetLegend>
    <i class="pi pi-user"></i> User <strong>Profile</strong>
  </span>
  <p>Body content</p>
</ui-lib-fieldset>

Variants

Three built-in design variants: material, bootstrap, and minimal.

@for (variant of variants; track variant) {

Content inside the {{ variant }} variant fieldset.

} 
<ui-lib-fieldset variant="material" legend="Material" />
<ui-lib-fieldset variant="bootstrap" legend="Bootstrap" />
<ui-lib-fieldset variant="minimal" legend="Minimal" />

Toggle Event

Listen to the (toggled) output to react when the panel opens or closes.

Toggle me to see the event fire.

 
<ui-lib-fieldset
  legend="Event Demo"
  [toggleable]="true"
  (toggled)="onToggle($event)"
></ui-lib-fieldset>

// component.ts
onToggle(event: FieldsetToggleEvent): void {
  console.log('collapsed:', event.collapsed);
}

API

Inputs

Input Type Default Description
legend string '' Text label rendered in the legend header.
toggleable boolean false Enables collapse/expand on legend click.
collapsed boolean false Current collapsed state. Supports [(collapsed)] two-way binding.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Falls back to global theme when null.
styleClass string | null null Additional CSS classes on the host element.

Outputs

Output Type Description
toggled FieldsetToggleEvent Emitted when the panel collapses or expands. Carries { collapsed: boolean }.

Content Projection

Slot Description
[fieldsetLegend] Custom HTML for the legend / header area.
default Body content rendered inside the panel.
