ARIA Attributes

SplitButton follows a menu-button pattern. The main action remains a standard button, while the trigger exposes menu semantics and the panel uses role="menu".

  • Main and menu buttons support Enter/Space; Escape closes the open menu.
  • Menu items use arrow-key navigation with Home/End and expose aria-disabled when needed.
  • Focus returns to the menu button when the menu closes from Escape or item activation.
  • This demo reports command results in a polite live region at the top of the page.
