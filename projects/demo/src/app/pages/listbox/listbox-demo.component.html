Listbox

A scrollable list of options with single or multiple selection, inline filtering, option groups, checkbox mode, and full keyboard navigation. Implements ControlValueAccessor for seamless ngModel and reactive-form integration.

a. Basic

Single-select listbox bound with ngModel. Click an option to select it; click the same option again to deselect.

Selected: {{ basicValue | json }}

b. Multiple Selection

Enable [multiple]="true" to let users pick several items. Each click toggles individual selection.

Selected: {{ multipleValue | json }}

c. Filter

Add [filter]="true" to show a search input above the list. Options are narrowed in real-time using the contains strategy by default.

Selected: {{ filterValue | json }}

d. Filter Match Modes

Control matching strategy with filterMatchMode: contains (default), startsWith, endsWith, or equals.

startsWith
endsWith

e. Option Groups

Pass a nested array and set [group]="true". Use optionGroupLabel and optionGroupChildren to map your data shape.

Selected: {{ groupValue | json }}

f. Checkbox Mode

Combine [multiple]="true" and [checkbox]="true" to render a checkbox beside each option for visual multi-select affordance.

Selected: {{ checkboxValue | json }}

g. Toggle All

Add [showToggleAll]="true" in multiple + checkbox mode to show a header checkbox that selects or deselects all non-disabled options.

Selected ({{ toggleAllValue.length }}): {{ toggleAllValue | json }}

h. Disabled

[disabled]="true" prevents all interaction. Individual options can also be disabled via an disabled: true field on the option object.
