ClassNames

A utility function and Angular pipe for composing CSS class strings from strings, conditional objects, and arrays — similar to the popular classnames npm package and PrimeNG's classNames utility.

Function Usage

Import classNames and call it from TypeScript. Strings, objects (keys with truthy values included), arrays, and falsy values (ignored) are all accepted.

Pipe Usage

Import ClassNamesPipe into your standalone component and use classNames as a template pipe. Pass additional arguments after the colon.

Interactive Playground

Toggle the flags below and watch the resolved class string update in real time.

active: {{ isActive() }} disabled: {{ isDisabled() }} highlighted: {{ isHighlighted() }}
classNames() result: {{ computedClasses() }}
pipe result: {{ { 'demo-box': true, 'demo-box--active': isActive(), 'demo-box--disabled': isDisabled(), 'demo-box--highlighted': isHighlighted() } | classNames }}

API Reference

classNames(...values)

Parameter Type Description
...values ClassNameValue[] One or more values. Strings are included as-is. Object keys with truthy values are included. Arrays are recursively processed. Falsy primitives are ignored.

ClassNamesPipe

Argument Type Description
value (primary) ClassNameValue The primary value passed before the pipe symbol.
...additional ClassNameValue[] Optional extra values passed as pipe arguments after :.
