Menubar

A horizontal navigation bar with dropdown submenus. Items can have nested sub-panels for hierarchical navigation. Supports start/end content projection, three visual variants, three sizes, and a responsive mobile toggle.

Basic

Top-level items with dropdown menus. Items with an items array open a single-column panel below the bar. 

{{ basicModel }}
<ui-lib-menubar [model]="basicModel" />

Nested Submenus

Sub-items with an items array open a nested panel to the right. Hover over an item with the › caret to see the nested panel.

Disabled Items

Use disabled: true on root or sub items to prevent interaction.

Command Callbacks

Attach a command function to any leaf item to react to activations.

Start / End Slots

Project content into the bar using the menubarStart and menubarEnd attribute selectors.

◆ MyApp 👤 Account 
<ui-lib-menubar [model]="items">
  <span menubarStart>MyApp</span>
  <button menubarEnd>Sign In</button>
</ui-lib-menubar>
