The DataGridComponent is the library's flagship premium component. It renders tabular data with a declarative column DSL, supports full client-side or server-side (lazy) operations, and renders only the visible rows when virtual scroll is enabled.

  • Virtual scroll — windowed rendering for 10,000+ rows
  • Column pinning — freeze columns to start or end
  • Column resizing — drag header edges in expand or fit mode
  • Cell editing — click-to-edit with custom editor templates
  • Lazy load — delegate sort/filter/page to a server API
  • Selection — single, multiple, checkbox, or radio
  • Sorting — single or multi-column (Ctrl+click)
  • Filtering — global filter and per-column filter inputs
