a. Basic

A simple labeled textarea bound with ngModel.

Value: "{{ basicValue }}"

b. Auto Resize

When autoResize is true the textarea grows in height automatically to fit its content.

c. Counter

Display a live character count below the textarea.

d. Max Length

Combine maxLength with showCounter to display a count / limit indicator.

e. Sizes

Three size tokens: sm, md, lg.

f. Variants

Three design variants: material, bootstrap, minimal.

g. Disabled

The textarea is non-interactive and visually dimmed.

h. Read-only

Focusable but not editable — content can be selected and copied.

i. Invalid

Pass an error string to show a validation message.

j. Reactive Forms

Full integration with Angular reactive forms including validation state and error display.
