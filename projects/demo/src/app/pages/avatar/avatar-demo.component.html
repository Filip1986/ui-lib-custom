Avatar

Represents a person or entity with an image, initials, or icon. Supports three shapes, three sizes, and three theme variants. Group multiple avatars with AvatarGroup for a stacked appearance.

Image

Provide an image URL to display a photo. Use imageAlt to set the accessible description.

Label (Initials)

When no image is available, set label to display initials or a short text.

Icon

Set icon to a PrimeIcons class string to render an icon inside the avatar.

Shapes

Use shape="circle" (default) or shape="square".

Variants

Three visual variants match the global theme: material, bootstrap, and minimal.

Avatar Group

Wrap multiple avatars in ui-lib-avatar-group to stack them with overlap. Set ariaLabel on the group for accessibility.

API Reference

Avatar Inputs

Input Type Default Description
image string | null null URL of the image to display.
imageAlt string 'Avatar' Alt text for the image (used as aria-label).
label string | null null Initials or short text shown when no image is set.
icon string | null null CSS class string for an icon (e.g. pi pi-user).
size 'sm' | 'md' | 'lg' 'md' Size of the avatar.
shape 'circle' | 'square' 'circle' Shape of the avatar.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Inherits global theme when null.
ariaLabel string | null null Overrides the auto-resolved accessible label.
styleClass string | null null Extra CSS classes to append to the host element.

AvatarGroup Inputs

Input Type Default Description
ariaLabel string | null null Accessible label for the group (role="group").
