Tooltip

An attribute directive that attaches a floating label to any host element. Triggered by hover or focus, supports four positions, show/hide delays, three design variants, and auto-flip when space is limited.

Directive Overlay A11y

Basic

Add [uiLibTooltip] with the label text to any element. The tooltip appears above the host by default.

Save Delete Share 
<button uiLibTooltip="Save the document">Save</button>

Positions

Use tooltipPosition to control placement: top, bottom, left, or right. The tooltip flips automatically if there is insufficient space in the requested direction.

Top Bottom Left Right 
<button uiLibTooltip="Top tooltip" tooltipPosition="top">Top</button>
<button uiLibTooltip="Bottom tooltip" tooltipPosition="bottom">Bottom</button>
<button uiLibTooltip="Left tooltip" tooltipPosition="left">Left</button>
<button uiLibTooltip="Right tooltip" tooltipPosition="right">Right</button>

Focus event

Set tooltipEvent="focus" to show the tooltip on keyboard focus instead of hover. Useful for form inputs to provide contextual guidance. Use "both" to support both hover and focus. 

<input uiLibTooltip="Enter your full name" tooltipEvent="focus" />
<input uiLibTooltip="Accepts hover and focus" tooltipEvent="both" />

Show & hide delays

Use showDelay and hideDelay (milliseconds) to debounce rapid mouse movements. Useful in dense UIs to avoid tooltip flicker.

400 ms show delay 600 ms hide delay Show 300 ms / Hide 400 ms 
<button uiLibTooltip="Appears after 400 ms" [showDelay]="400">Hover</button>

Disabled

Set [tooltipDisabled]="true" to suppress the tooltip entirely. Bind to a signal for runtime toggling.

{{ tooltipDisabled() ? 'Tooltip disabled' : 'Tooltip enabled' }} Toggle disabled 
<button uiLibTooltip="Hidden" [tooltipDisabled]="isDisabled">Hover</button>

Variants

Three design variants are available: material, bootstrap, and minimal. When tooltipVariant is not set, the variant is inherited from ThemeConfigService.

Material Bootstrap Minimal 
<button uiLibTooltip="Material tooltip" tooltipVariant="material">Material</button>

Long text

The tooltip wraps text at 14rem max width and breaks long words automatically.

Hover for long text

API

Selector: [uiLibTooltip]

Input Type Default Description
uiLibTooltip string '' The tooltip label text.
tooltipPosition 'top' | 'bottom' | 'left' | 'right' 'top' Preferred position. Flips automatically when space is lacking.
tooltipEvent 'hover' | 'focus' | 'both' 'hover' Which host events trigger the tooltip.
showDelay number 0 Milliseconds to wait before showing.
hideDelay number 0 Milliseconds to wait before hiding.
tooltipDisabled boolean false Prevent the tooltip from showing.
tooltipVariant TooltipVariant | null null Design variant. Falls back to ThemeConfigService.
