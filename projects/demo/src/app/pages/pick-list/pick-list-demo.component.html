PickList

Transfers items between a source list and a target list with reorder controls, optional filter inputs, drag-and-drop support, and full keyboard accessibility.

1. Basic

Use [(source)] and [(target)] two-way bindings to let the component manage transfers. trackBy identifies items by a property key.

2. Filter

Set filterBy to show independent filter inputs above each list. Filtering never mutates the underlying arrays.

3. Custom Templates

Supply uiPickListSourceHeader, uiPickListTargetHeader, uiPickListItem, and uiPickListEmpty directives for fully custom rendering. The item template receives let-item as its implicit context.

4. Drag & Drop

Enable [dragDrop]="true" to allow dragging items within a list to reorder them, or dragging between lists to transfer them. A drop indicator line shows the insertion point.

5. Accessibility

Keyboard shortcuts

KeyAction
Arrow Down / UpNavigate items in the focused list.
Space / EnterToggle selection of focused item.
Ctrl+ASelect all visible items in the focused list.
EscapeClear selection in the focused list.
Ctrl+→Transfer selected source items to target.
Ctrl+←Return selected target items to source.
Ctrl+↑Move selected items up within their list.
Ctrl+↓Move selected items down within their list.
Ctrl+HomeMove selected items to top of list.
Ctrl+EndMove selected items to bottom of list.

ARIA

  • Each list: role="listbox", aria-multiselectable="true".
  • Each item: role="option", aria-selected.
  • Transfer and reorder buttons carry descriptive aria-labels.
  • All transfers and reorders are announced via LiveAnnouncerService.
  • Provide a label with [sourceAriaLabel] / [targetAriaLabel].
