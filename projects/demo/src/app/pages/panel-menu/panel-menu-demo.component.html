PanelMenu

An accordion-style, hierarchical navigation menu driven by a data model. Root items act as collapsible panels; leaf items are directly activatable.

Basic

Panel headers expand on click to reveal their child items.

<ui-lib-panel-menu [model]="items" />

Initially Expanded

Set expanded: true on a root item to have it open on first render. 

items = [
  { label: 'File', expanded: true, items: [...] },
  { label: 'Edit', items: [...] },
]

Multiple Mode

By default only one panel can be open at a time. Set [multiple]="true" to allow multiple expanded panels simultaneously. 

<ui-lib-panel-menu [model]="items" [multiple]="true" />

Nested Sub-Items

Items within a panel can also have their own items array, creating deeply nested collapsible groups.

Disabled Items

Set disabled: true on any item — root panels or leaf items — to prevent interaction.

URL Items

Provide url on a leaf item to render it as an anchor tag. Use target for link targeting. 

{ label: 'Angular Docs', url: 'https://angular.dev', target: '_blank' }

Variants

Supports three design-system variants: material, bootstrap, and minimal. Falls back to the global ThemeConfigService variant when not set.

Events

(itemClick) fires when a leaf item is activated. (panelToggle) fires when a root panel expands or collapses.

API

PanelMenu Inputs

Name Type Default Description
model PanelMenuItem[] [] Array of root-level menu items.
multiple boolean false Allow multiple root panels open at once.
variant 'material' | 'bootstrap' | 'minimal' | null null Design variant. Falls back to ThemeConfigService.
size 'sm' | 'md' | 'lg' 'md' Size token.
styleClass string | null null Extra CSS class on the host element.
ariaLabel string 'Panel Menu' Accessible label for the root container.

PanelMenu Outputs

Name Type Description
(itemClick) PanelMenuCommandEvent Emitted when a leaf item is activated.
(panelToggle) PanelMenuPanelToggleEvent Emitted when a root panel expands or collapses.

PanelMenuItem Interface

Property Type Description
label string Display text.
icon string Icon CSS class rendered before the label.
expanded boolean Initial expansion state (root items only).
disabled boolean Prevents interaction.
separator boolean Renders a visual divider.
visible boolean Hides the item when false.
items PanelMenuItem[] Child items — makes this item a collapsible group.
command (event) => void Callback invoked when a leaf item is activated.
url string Renders the item as an anchor link.
target string Link target attribute (e.g. '_blank').
styleClass string Extra CSS class on the item element.
