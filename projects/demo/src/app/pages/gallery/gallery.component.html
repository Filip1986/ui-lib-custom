Basic

A gallery with item and thumbnail templates. Navigation arrows and a scrollable thumbnail strip are shown by default.

With Caption

Use the #galleriaCaption template slot to overlay a caption on the active item.

Indicators

Replace the thumbnail strip with dot indicators by enabling showIndicators and setting showThumbnails to false.

Thumbnail Position

The thumbnail strip can be positioned on any side using thumbnailsPosition: bottom (default), top, left, or right.

@for (pos of positionOptions; track pos) { {{ pos }} }

Indicators on Item

Set showThumbnails="false", showIndicators="true", and showIndicatorsOnItem="true" to overlay dots directly on the image.

Fullscreen

Enable fullScreen to show a maximize button. Clicking it opens the gallery in a full-screen overlay. The visible two-way binding controls open/close state.

{{ image.title }}

Circular & Autoplay

With circular="true" navigation wraps around. Adding autoPlay="true" advances the gallery automatically every 3 seconds.

Variants

Three design variants are available: material, bootstrap, and minimal.

@for (v of variantOptions; track v) { {{ v }} }

Sizes

The size input (sm / md / lg) controls the thumbnail strip sizing.

@for (s of sizeOptions; track s) { {{ s }} }
