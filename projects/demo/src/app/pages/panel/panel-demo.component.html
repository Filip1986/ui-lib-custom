Panel

A flexible content container with an optional collapsible body, header actions, and footer slot. Supports three design variants and a smooth CSS grid-row collapse animation.

import { Panel } from 'ui-lib-custom/panel';

Basic

A simple panel with a header string and body content.

The Panel component is a lightweight wrapper that groups related content under a labelled header. It renders as a bordered card with a header area, body, and optional footer. 

<ui-lib-panel header="Introduction">
  <p>Panel body content.</p>
</ui-lib-panel>

Toggleable

Set [toggleable]="true" to display a chevron button in the header that collapses or expands the body with a smooth CSS animation.

This content is collapsible. Click the chevron button in the header to toggle visibility. The collapse animation uses the CSS grid-template-rows technique for a smooth height transition without JavaScript.

The two-way [(collapsed)] binding keeps the parent in sync with the current state, and the (toggled) output emits a PanelToggleEvent on every change.

<ui-lib-panel
  header="Collapsible Section"
  [toggleable]="true"
  [(collapsed)]="isCollapsed"
  (toggled)="handleToggle($event)"
>
  <p>Panel body content.</p>
</ui-lib-panel>

Collapsed by Default

Pass [collapsed]="true" (or initialise the model signal as true) to start the panel in collapsed state.

This panel starts collapsed. Click the chevron to reveal the content.

 
<ui-lib-panel header="Advanced Options" [toggleable]="true" [collapsed]="true">
  <p>Hidden until expanded.</p>
</ui-lib-panel>

Custom Header

Project rich HTML into the header using the [panelHeader] attribute selector. This replaces the plain text header.

★ Featured New

The custom header slot lets you place any HTML — icons, badges, status indicators — into the panel title area. 

<ui-lib-panel [toggleable]="true">
  <span panelHeader>
    <strong>Featured</strong>
    <span class="badge">New</span>
  </span>
  <p>Body content.</p>
</ui-lib-panel>

Header Icons

Project action buttons or icons into the right side of the header using the [panelIcons] attribute selector. The icons appear before the built-in toggle button.

Panel body with custom action icons in the header.

 
<ui-lib-panel header="Report" [toggleable]="true">
  <button panelIcons type="button">Refresh</button>
  <p>Body content.</p>
</ui-lib-panel>

Footer

Project content into the footer using the [panelFooter] attribute selector. The footer renders below the body and is always visible regardless of collapsed state.

By using this service you agree to the terms outlined below. Please read carefully before proceeding with registration or purchase. 

<ui-lib-panel header="Terms of Service">
  <p>Body content.</p>
  <div panelFooter>
    <button type="button">Decline</button>
    <button type="button">Accept</button>
  </div>
</ui-lib-panel>

Variants

Three design variants: material, bootstrap, and minimal.

Rounded corners with a subtle box shadow. Toggle button is circular.

Flat styling with a 4 px border radius and bold border.

Minimal styling with no shadow and instant toggle (0 ms transition).

 
<ui-lib-panel header="Panel" variant="material" />
<ui-lib-panel header="Panel" variant="bootstrap" />
<ui-lib-panel header="Panel" variant="minimal" />

API

Inputs

InputTypeDefaultDescription
headerstring''Header title text.
toggleablebooleanfalseWhether the body can be collapsed.
collapsedboolean (model)falseTwo-way bound collapsed state.
variant'material' | 'bootstrap' | 'minimal' | nullnullDesign variant. Falls back to ThemeConfigService.
styleClassstring | nullnullExtra CSS classes added to the host element.

Outputs

OutputTypeDescription
toggledPanelToggleEventEmitted when the collapsed state changes. Carries collapsed boolean and optional originalEvent.

Content Projection

SlotSelectorDescription
Body(default)Main body content.
Custom header[panelHeader]Replaces the text header area.
Header icons[panelIcons]Action buttons in the header right area.
Footer[panelFooter]Footer content below the body.
