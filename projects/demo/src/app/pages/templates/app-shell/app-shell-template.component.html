Overview

The App Shell template provides the structural bones every application needs: a sticky top navigation bar, a collapsible sidebar with grouped navigation, and a scrollable main content area. Copy the pattern into your project and start customising immediately.

Sticky Topbar

Fixed top navigation with logo, search, notifications, and user avatar.

Collapsible Sidebar

Grouped navigation links with active-state highlighting and badge support.

Dashboard-ready

Pre-wired stat cards, a data panel, and a recent-activity feed section.

Fully Themed

Every colour and spacing value comes from --uilib-* CSS variables.
