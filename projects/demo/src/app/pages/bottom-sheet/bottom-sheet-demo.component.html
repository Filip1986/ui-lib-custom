BottomSheet

A slide-up overlay panel anchored to the bottom of the viewport. Ideal for contextual actions, share menus, and quick forms on mobile and desktop. Includes a focus trap, focus restoration on close, scroll locking, and full keyboard + screen-reader support.

Open Sheet

Basic

Bind [(visible)] to control the sheet. Click the backdrop or press Escape to dismiss. Sheet instances live outside the layout wrapper so they render at the correct stacking context.

Open Sheet

With Header

Pass the header input to render a title bar with an accessible close button. The panel gets aria-labelledby pointing to the title automatically.

Variants

Three design variants adjust the panel shape and shadow. When variant is null (default), it inherits from ThemeConfigService.

@for (variant of variants; track variant) { {{ variant }} }

Inputs

Input Type Default Description
visible boolean false Controls open/close state. Two-way bindable via [(visible)]. A model() signal internally.
header string '' Optional header text. When set, renders a title bar with a close button and wires aria-labelledby.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Inherits from ThemeConfigService when null.
showBackdrop boolean true Whether to show the semi-transparent backdrop behind the panel.
closeOnBackdrop boolean true Whether clicking the backdrop closes the sheet.
closeOnEscape boolean true Whether pressing Escape closes the sheet.
styleClass string | null null Additional CSS classes applied to the host element.

Outputs

Output Payload Description
(shown) void Emits after the sheet transitions to open.
(hidden) void Emits after the sheet transitions to closed.

Content Slots

Selector Description
default Main body content rendered inside a scrollable content area.
[bottomSheetFooter] Sticky footer pinned to the bottom of the panel. Hidden when empty.

ARIA Attributes

Element Attribute Notes
Panel role="dialog" Marks the sheet as a dialog region.
Panel aria-modal="true" Signals to screen readers that content behind the sheet is inert (only when open).
Panel aria-labelledby References the header title element when header is provided.
Panel tabindex="-1" Enables programmatic focus on the panel element itself on open.
Host / Backdrop aria-hidden="true" Hides the component and backdrop from screen readers when closed.
Close button aria-label="Close" Accessible name for the icon-only close button.
Close icon SVG aria-hidden="true" Decorative — the button's aria-label provides the accessible name.

Keyboard Interaction

Key Action
Escape Closes the sheet (when [closeOnEscape]="true") and returns focus to the previously focused element.
Tab Cycles focus forward through all focusable elements inside the panel. Wraps from last to first.
Shift+Tab Cycles focus backward. Wraps from first to last.

Notes

  • Focus trap: When the sheet opens, focus is moved inside the panel and Tab / Shift+Tab are trapped within it via the FocusTrap utility from ui-lib-custom/core.
  • Focus restoration: When the sheet closes, focus automatically returns to the element that was focused before the sheet opened.
  • Scroll is locked on <body> while the sheet is open.
  • All animations respect prefers-reduced-motion: reduce — transitions are disabled when the user prefers reduced motion.
