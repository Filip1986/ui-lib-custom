BottomSheet

A slide-up overlay panel anchored to the bottom of the viewport. Ideal for contextual actions, share menus, and quick forms on mobile and desktop.

Basic

Bind [(visible)] to control the sheet. Click the backdrop or press Escape to dismiss. 

<ui-lib-bottom-sheet [(visible)]="isOpen">
  <p>Sheet content goes here.</p>
</ui-lib-bottom-sheet>

With Header

Pass header to render a title bar with a close button. 

<ui-lib-bottom-sheet [(visible)]="isOpen" header="Share">
  <p>Choose where to share.</p>
</ui-lib-bottom-sheet>

Footer Slot

Project an element with the [bottomSheetFooter] attribute for a sticky action area pinned to the bottom of the panel. 

<ui-lib-bottom-sheet [(visible)]="isOpen" header="Confirm">
  <p>Are you sure you want to proceed?</p>
  <div bottomSheetFooter>
    <button (click)="isOpen.set(false)">Cancel</button>
    <button (click)="confirm()">Confirm</button>
  </div>
</ui-lib-bottom-sheet>

Disable Backdrop Dismiss

Set [closeOnBackdrop]="false" to prevent clicking the backdrop from closing the sheet.

Variants

Three design variants adapt the panel shape and shadow to your chosen design system: material, bootstrap, and minimal.

@for (variant of variants; track variant) { }

API

Inputs

Input Type Default Description
visible model<boolean> false Controls open/close state. Supports two-way binding via [(visible)].
header string '' Optional header text. When set, renders a title bar with a close button.
variant 'material' | 'bootstrap' | 'minimal' | null null Visual variant. Inherits from ThemeConfigService when null.
showBackdrop boolean true Whether to show the semi-transparent backdrop behind the panel.
closeOnBackdrop boolean true Whether clicking the backdrop closes the sheet.
closeOnEscape boolean true Whether pressing Escape closes the sheet.
styleClass string | null null Additional CSS classes applied to the host element.

Outputs

Output Type Description
shown void Emits when the sheet transitions to open.
hidden void Emits when the sheet transitions to closed.

Content Projection

Selector Description
(default) Main body content rendered inside a scrollable content area.
[bottomSheetFooter] Sticky footer content pinned to the bottom of the panel.
